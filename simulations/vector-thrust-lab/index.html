纵列双发矢量推力布局 — 高保真 6-DOF 动力学模型

前电机（+a, 0, 0）正转、绕 zb 轴摆动（偏航）；尾电机（−b, 0, 0）反转、绕 yb 轴摆动（俯仰）。两电机轴线在零摆角时均过质心。推力 T = kT·ω²，反扭矩 τ = kQ·ω² + Jp·ω̇。

推力通道六维力 / 力矩（机体系）

F_x = T_f·cosδ_f + T_t·cosδ_t    F_y = T_f·sinδ_f    F_z = −T_t·sinδ_t
M_x = −τ_f·cosδ_f + τ_t·cosδ_t
M_y = −b·T_t·sinδ_t − τ_f·sinδ_f
M_z = a·T_f·sinδ_f − τ_t·sinδ_t

刚体动力学方程

m·v̇ = F推力 + F气动 + m·gb − m·ω×v
I·ω̇ = M推力 + M气动 − ω×(I·ω) − ω×h转子   ·   q̇ = ½·q⊗ωb

气动模型（风轴系, q̄ = ½ρV²）

L = q̄·S·CLα·α    D = q̄·S·(CD0 + k·CL²)    Y = q̄·S·CYβ·β
Mx,a = q̄·S·b·(Clβ·β + Clp·p̂)
My,a = q̄·S·c̄·(Cm0 + Cmα·α + Cmq·q̂)
Mz,a = q̄·S·b·(Cnβ·β + Cnr·r̂)

阻尼导数 Cmq / Clp / Cnr 提供物理角速率阻尼：摆角阶跃 → 角加速度 → 角速度被气动阻尼衰减、α/β 建立后被静稳定力矩平衡 —— 响应有界，无需任何人工阻尼。低速时气动阻尼 ∝ q̄ 趋零，姿态稳定转由 SAS 闭环承担。

SAS 增稳（角速率 + 姿态角比例/积分反馈, 反馈极性按控制效率符号整定）

δ_t ← δ_t + kq·q + kθ·θ + ki∫θ   δ_f ← δ_f − kr·r    Δω ← Δω + kp·p + kφ·φ + kiφ∫φ
