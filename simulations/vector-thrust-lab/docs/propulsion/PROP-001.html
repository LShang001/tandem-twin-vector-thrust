PROP-001 推进与摆座模型

本文定位「纵列双发矢量推力飞行器」推进系统的工程实现模型，覆盖电机-螺旋桨集总参数、一阶转速滞后、前后单轴摆座运动学及六维力/力矩映射。对应源码实现见 simulations/vector-thrust-lab/src/core/propulsion.mjs，理论背景见 THY-002。

1 模型范围

当前推进模型为电机-螺旋桨集总模型，包含：

  • 螺旋桨推力 T = kT·ω²；
  • 传给机体的反扭矩 Q = kQ·ω² + Jp·ω̇；
  • 电机转速一阶滞后；
  • 前/后电机单轴摆座的六维力/力矩映射。

未包含：电调开关动态、电池内阻与压降、电机热模型、螺旋桨前进比效应、真实舵机位置/速率/加速度动态。这些理想化声明见 §6。

2 推力与反扭矩模型

2.1 基本公式

T = kT · ω²                         （螺旋桨推力）
Q = kQ · ω² + Jp · ω̇                （传给机体的反扭矩）
  • kT：推力系数，量纲 N·s²；
  • kQ：反扭系数，量纲 N·m·s²；
  • Jp：桨+转子转动惯量，量纲 kg·m²；
  • ω̇ 项为转子加速时经电机定子传给机体的附加反扭。

2.2 参数表

参数 值 单位 含义 来源
kT 2.2×10⁻⁵ N·s² 推力系数 MODEL-DEFAULT
kQ 1.6×10⁻⁶ N·m·s² 反扭系数 MODEL-DEFAULT
Jp 2.0×10⁻⁴ kg·m² 桨+转子转动惯量 MODEL-DEFAULT
wMax 900 rad/s 最大转速 MODEL-DEFAULT

稳态（ω̇=0）时 Q = kQ·ω²。以 wMax=900 rad/s 为例：

T_max = kT · wMax² = 2.2×10⁻⁵ × 900² ≈ 17.8 N
Q_max = kQ · wMax² = 1.6×10⁻⁶ × 900² ≈ 1.30 N·m

双发全油门总推力约 35.6 N，对应整机 2.6 kg 约 1.4g 推力裕度。

2.3 反扭的物理来源

见 THY-002 §3：真正传给机体的是电机电磁扭矩 τ_e = kQ·ω² + Jp·ω̇，而非纯气动阻力矩。加速时 Jp·ω̇ 与 kQ·ω² 同号，机体承受更大反扭；减速时惯性项反向。该效应在滚转阶跃机动中不可忽略。

3 电机一阶滞后模型

3.1 离散更新公式

propulsion.mjs:15–16：

S.wf += (wfT - S.wf) * Math.min(dt / P.tauM, 1);
S.wt += (wtT - S.wt) * Math.min(dt / P.tauM, 1);

符号表达：

ω(k+1) = ω(k) + (ω_target − ω(k)) · min(dt / τM, 1)

3.2 min(dt/τM, 1) 截断语义

  • 当 dt < τM 时，为显式欧拉近似的一阶惯性环节；
  • 当 dt ≥ τM 时，截断为 1，转速直达目标值，避免大步长下欠阻尼振荡或数值不稳定；
  • 该行为由 propulsion.test.mjs:56 测试记录（用 100 个小步逼近连续一阶响应，验证一个时间常数后达 63.2%）。

3.3 参数

参数 值 单位 含义 来源
tauM 0.28 s 电机一阶滞后时间常数 MODEL-DEFAULT

4 摆座运动学

4.1 坐标系与摆轴

  • 机体系 {b}：x_b 指机头，y_b 指右翼，z_b 指下方，原点在质心；
  • 前电机位于 r_f = [+a, 0, 0]，绕 z_b 单轴摆动，摆角 δ_f；
  • 后电机位于 r_t = [−b, 0, 0]，绕 y_b 单轴摆动，摆角 δ_t；
  • 零摆角时两电机推力线均通过质心，静态推力不产生俯仰/偏航力矩。

4.2 旋转矩阵

        ┌ cosδ_t   0   sinδ_t ┐              ┌ cosδ_f  -sinδ_f   0 ┐
R_y =   │   0      1     0    │      R_z =   │ sinδ_f   cosδ_f   0 │
        └-sinδ_t   0   cosδ_t ┘              └   0        0      1 ┘

电机系推力矢量 T·[1,0,0]ᵀ 投影到机体系：

F_f = R_z(δ_f)·[T_f,0,0]ᵀ = [T_f·cosδ_f,  T_f·sinδ_f,  0        ]ᵀ
F_t = R_y(δ_t)·[T_t,0,0]ᵀ = [T_t·cosδ_t,  0,          -T_t·sinδ_t]ᵀ

5 六维力/力矩映射

5.1 完整公式表

propulsion.mjs:28–33 给出的机体系合力与对质心力矩：

分量 公式 物理来源
F_x T_f·cosδ_f + T_t·cosδ_t 两推力沿 x_b 投影之和
F_y T_f·sinδ_f 前推力沿 y_b 侧向分量
F_z −T_t·sinδ_t 尾推力沿 z_b 法向分量
M_x −Q_f·cosδ_f + Q_t·cosδ_t 前后反扭沿 x_b 投影之差
M_y −b·T_t·sinδ_t − Q_f·sinδ_f 尾推力俯仰力矩 + 前反扭耦合
M_z a·T_f·sinδ_f − Q_t·sinδ_t 前推力偏航力矩 − 尾反扭耦合

5.2 符号逐项核对

  • F_x：前/尾推力均主要沿 +x_b，故同为正；
  • F_y：仅前摆角产生 +y_b 侧向力，sinδ_f 为正时 F_y > 0；
  • F_z：尾摆角使推力偏离 x_b 指向 +z_b（向下），故机身受 −z_b 力；
  • M_x：前电机正转，机体承受 −x_b 反扭；后电机反转，机体承受 +x_b 反扭；
  • M_y：尾推力线偏离 x_b 产生低头力矩（−y_b）；前反扭投影到 y_b 为 −Q_f·sinδ_f；
  • M_z：前推力侧向分量对质心产生 +z_b 力矩；尾反扭投影到 z_b 为 −Q_t·sinδ_t。

5.3 小角度线性化

力矩 线性化表达式 主控变量 效率符号
M_x −2·kQ·ω0²·Δω Δω 负
M_y −b·T_t·δ_t δ_t 负
M_z a·T_f·δ_f δ_f 正

其中 ω0 = thr·wMax，T_f = T_t = kT·ω0²。

6 差速分配与 wMax 边界讨论

6.1 差速分配律

ω0 = thr · wMax
ω_f,target = ω0 · √(max(0, 1 + Δω_cmd))
ω_t,target = ω0 · √(max(0, 1 − Δω_cmd))

6.2 超测算例

当前实现无 wMax 饱和保护：差速大开度时目标转速可超过 wMax。

例：thr=1，Δω_cmd=0.55：

ω_f,target = 900 · √(1+0.55) ≈ 900 · 1.245 ≈ 1121 rad/s  >  wMax
ω_t,target = 900 · √(1−0.55) ≈ 900 · 0.671 ≈  604 rad/s

该超调在当前代码中被如实计算，模型本身不对转速做物理上限截断。这意味着：

  • 大差速指令下电机模型输出高于额定最大转速；
  • 滚转力矩 M_x ∝ (ω_t² − ω_f²) 仍按开方律解析增长；
  • 真实电机/螺旋桨在超转速时会进入非线性区，甚至结构失效，但当前模型未反映。

6.3 边界约束

Math.max(0, 1 ± Δω) 仅保证根号内非负，即：

|Δω_cmd| ≤ 1

实际 SAS 输出已限幅至 |Δω| ≤ dwMax = 0.7，UI 输入限幅至 |dw| ≤ dwUiMax = 0.55，均小于 1，根号内恒正。

7 当前理想化声明

理想化项 当前处理 备注
摆角动态 理想瞬时到位 无位置、速率、加速度限制
转速饱和保护 无 差速大开度可超 wMax
前进比效应 无 推力/反扭系数不随空速变化
电调/电池动态 无 电机直接响应目标转速
电机热模型 无 允许持续超转速（模型层面）
螺旋桨滑流干扰 无 前后桨无气动干扰

上述简化使模型适用于概念级六自由度仿真，但不代表真实飞行性能。

8 待标定项与台架标定计划

8.1 待标定参数

参数 当前状态 标定方法设想

8.2 状态升级路径

按 GOV-002 §3.3 参数状态分类：

MODEL-DEFAULT  →  BENCH-TESTED

升级条件：

  1. 完成台架试验并记录原始数据；
  2. 将拟合过程、不确定度、试验条件写入 docs/registers/参数数据手册.md；
  3. 更新 models/aircraft-model.json 中对应参数的 source 与 confidence；
  4. 运行 python tools/sync-params.py 重新生成 parameters.mjs；
  5. 运行 node --test tests/*.test.mjs 确认测试通过或按需更新基线。

8.3 标定优先级

  1. tauM：直接影响滚转阶跃响应与 SAS 稳定性，优先级最高；
  2. kT：决定推力包线与配平油门，优先级高；
  3. kQ / Jp：影响滚转通道效率与机动耦合，优先级中。

10 成熟度声明

  • 推进模型为概念级集总模型，所有参数状态 MODEL-DEFAULT；
