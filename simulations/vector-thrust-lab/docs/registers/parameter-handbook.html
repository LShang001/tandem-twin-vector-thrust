8 使用说明

  1. 如需修改参数，请编辑 models/aircraft-model.json 后运行同步脚本；禁止直接修改 src/core/parameters.mjs。
  2. 参数从 MODEL-DEFAULT 升级至 ASSUMED / ESTIMATED / DATASHEET / BENCH-TESTED / FLIGHT-IDENTIFIED 时，须同步更新本手册的「来源」与「置信度」字段，并按 GOV-002 记录决策依据。
  3. 本手册未包含参数之间的耦合关系；设计研究时应结合 模型-代码追溯矩阵 与 假设与局限日志 使用。
