假设与局限日志

本日志记录「纵列双发矢量推力飞行器」C0 仿真模型中的所有已知假设、简化与局限。每项均说明其影响域、对结果可信度的影响评估、解除条件或改进方向，以及关联文档。新增假设应经变更评审后按顺序编号加入。

编号 假设 / 局限 影响域 影响评估 解除条件 / 改进方向 关联文档
ASM-001 点质量刚体：仅考虑质心平动与绕质心转动，无弹性变形、无附加质量、无惯量积 动力学、配平、模态分析 概念级分析可接受；高机动或柔性机体时会低估耦合与振动 引入结构有限元或多体模型，通过台架 / 模态试验更新惯量张量 MOD-001、src/core/dynamics.mjs
ASM-002 理想执行机构：前 / 尾摆角与差速指令瞬时到位，无舵机速率 / 带宽 / 间隙模型 控制、推进、频域分析 SAS 增益按理想执行机构整定，真实系统中高频相位损失可能导致振荡 增加执行机构一阶滞后或速率饱和模型；当前 REQ-CTL-003 列为未满足 CTL-001、REQ-001 REQ-CTL-003
ASM-003 wMax 饱和后不重分配：高转速侧已硬限幅，但低转速侧仍按原差速下降 推进、滚转通道安全 高油门大差速时总推力下降；油门 1.0、差速 0.55 时当前模型下降 27.5% 增加饱和感知重分配与遥测告警，并验证滚转优先/总推力优先策略 CTL-002、src/core/propulsion.mjs
ASM-004 线性小迎角气动：升力、侧力、力矩均为迎角 / 侧滑角的线性函数，无失速、无动态入流 气动、包线分析 仅适用于小迎角稳定飞行；大迎角 / 深失速 / 尾旋不可评估 引入失速模型、涡格法或 CFD 数据库；新增气动力_lookup MOD-001、AER-001
ASM-005 无风 / 大气扰动：空气静止、密度恒定 rho = 1.225 kg/m³，无风切变、突风、湍流 气动、控制鲁棒性 无法评估风扰下的姿态保持与航迹控制性能 增加恒定风、Dryden / Von Karman 湍流模型及大气密度随高度变化 src/core/aerodynamics.mjs、PROP-001
ASM-006 推力线过质心：零摆角时前后电机推力线均通过 CG，无静态推力力矩 推进、配平、稳定性 简化了配平计算；真实安装误差会产生附加力矩 测量真实推力线相对 CG 的偏移并引入安装矩阵 CFG-000、src/core/propulsion.mjs
ASM-007 旋翼陀螺效应简化：仅保留 Jp·ω̇ 与进动项 ω × h_转子，未完整建模陀螺力矩 / 挥舞 / 锥进 转动动力学、高机动响应 中低速小机动下可接受；快速滚转 / 俯仰组合机动时耦合精度有限 建立完整旋翼动力学或多桨叶坐标模型 THY-002、src/core/dynamics.mjs
ASM-008 渲染非尺度化：三维模型几何臂长 ±1.78 m 为示意性可视化尺寸，不等于物理力臂 a = b = 0.62 m UI / 可视化 不影响物理计算，但可能误导观众对尺寸与力臂的直观判断 文档化渲染尺度；可选提供按真实尺寸绘制的辅助模型 MOD-002、README
ASM-009 复位为热配平复位：完整复位将双电机直接恢复到配平转速，不模拟从静止冷启动 UI / 状态初始化 演示起点可重复，但不代表上电启动过程 若研究启动瞬态，新增独立“冷启动”场景，不复用配平复位 src/core/state.mjs、VER-001
ASM-010 欧拉 ±90° 奇异未处理：采用 3-2-1 欧拉角提取，俯仰角接近 ±90° 时出现万向锁 显示、SAS、HUD 大俯仰机动时欧拉角不连续，SAS 可能基于错误姿态角输出 限制演示机动范围；或在控制律中改用四元数直接反馈 THY-001、src/core/math.mjs
ASM-011 无传感器模型：角速度、姿态、位置、速度均直接可用，无噪声、无延迟、无失效 控制、导航、估计 无法验证状态估计算法、滤波器及传感器失效保护 增加 IMU、GPS、空速管模型及相应滤波器 CTL-001
ASM-012 无电调 / 电池动态：电机响应仅由一阶滞后 tauM 描述，忽略电流环、电压跌落、热衰减 推进、能源管理 瞬态推力与真实电调-电池系统存在差异 增加电调电流环、电池内阻与 SOC 热模型 PROP-001
ASM-013 增益手工整定未经辨识：SAS 增益为手工试凑值，未通过系统辨识或试飞验证 控制品质、稳定性裕度 所列工况内响应有界，但无定量稳定 / 带宽 / 鲁棒裕度 系统辨识获取真实模型，用经典 / 现代控制方法重新设计 CTL-001
ASM-014 低油门滚转退化：总油门低时，差速产生的反扭差减小，滚转控制能力下降 控制分配、机动能力 小油门下滚转响应变慢，大滚转指令可能饱和 设计油门-差速补偿或非线性控制分配律 CTL-002、src/core/propulsion.mjs
ASM-015 当前仅完成纵向三方程配平：aTrim、thrTrim、dtTrim 只闭合 F_x、F_z、M_y；尾摆反扭投影留下小幅 M_z 残差 配平、横航向响应、线性化 10 s 内纵向参考量保持良好，但仍出现约 0.168° 航向漂移和 0.275 m 侧移；不能直接作为完整状态 Jacobian 的平衡点 联立求解前摆、差速、侧滑和滚转姿态，使六分量力 / 力矩及运动学残差同时闭合 MOD-003、VER-001
