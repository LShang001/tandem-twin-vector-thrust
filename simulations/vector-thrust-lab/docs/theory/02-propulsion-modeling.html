纯理论推导（二）：推进系统的物理建模

本篇从第一性原理推导螺旋桨–电机推进单元的集总模型：桨盘动量理论 → k_T/k_Q 系数的物理来源 → 反扭矩的角动量守恒严格推导 → 电机电磁动态 → 旋转部件的陀螺力矩。与任何具体仿真实现无关。

1. 桨盘动量理论（Actuator Disk Theory）

1.1 控制体分析

将螺旋桨理想化为无限薄桨盘（面积 A = πD²/4），气流无黏不可压，盘上压强跳变但速度连续。取远前方截面 0、盘前盘后截面与远后方尾流截面 2，对流经管状控制体的流体分别列质量守恒与轴向动量定理：

流量:      ṁ = ρ·A·v_p                  （v_p 为盘面处流速）
推力:      T = ṁ·(v₂ − v₀)              （动量流率差）

1.2 盘面前后流速关系（Froude 引理）

对盘前（0→盘）与盘后（盘→2）两段分别应用伯努利方程（压强在盘上跳变 Δp，穿过盘面速度连续），联立消去 Δp 得：

T = Δp·A = ½·ρ·A·(v₂² − v₀²)  =  ρ·A·v_p·(v₂ − v₀)

两式比较即得 v_p = (v₀ + v₂)/2——盘面流速是远前方与远后方流速的平均。定义诱导速度 vᵢ = v_p − v₀，则 v₂ = v₀ + 2vᵢ。

1.3 静推（悬停）结果

v₀ = 0 时 v₂ = 2vᵢ，代入推力式：

T = 2·ρ·A·vᵢ²        ⟹        vᵢ = √(T / 2ρA)
理想诱导功率:  Pᵢ = T·vᵢ = T^(3/2) / √(2ρA)

物理推论：同样推力下，桨盘越大（A 越大），所需功率按 P ∝ T^(3/2)/√A 下降——大直径低转速优于小直径高转速。这是推进系统选型最基本的权衡。

1.4 前飞修正

v₀ > 0 时 T = 2ρA(v₀ + vᵢ)vᵢ，诱导速度随前飞速度单调下降：拉力式螺旋桨的”静推最大、随速度衰减”特性由此而来。该式是推力-速度曲线的理论骨架。

2. 从叶素到集总系数：k_T、k_Q 的来源

2.1 量纲分析

对几何相似的螺旋桨，在相似的进距比 J = V/(nD) 下，无量纲推力系数 C_T 与扭矩系数 C_Q 为常数：

C_T = T / (ρ·n²·D⁴)          C_Q = Q / (ρ·n²·D⁵)

n 为每秒转数。以角速度 ω = 2πn 表示：

T = (C_T·ρ·D⁴/4π²)·ω² = k_T·ω²        Q = (C_Q·ρ·D⁵/4π²)·ω² = k_Q·ω²

k_T ∝ ρD⁴、k_Q ∝ ρD⁵——系数不是经验黑箱，而是由桨叶气动外形与几何决定的相似律。Q/T 比值量级为 D·(C_Q/C_T)，即反扭矩与推力之比正比于桨径，这在评估滚转控制权限（差速反扭 vs 推力力臂）时是关键量级。

2.2 叶素理论要点（系数的气动来源）

将桨叶展向微元视为二维翼型，微元来流为旋转线速度 ωr 与轴向诱导速度 v₀+vᵢ 的合成，迎角 α(r) = 桨距角 φ(r) − 流入角。微元升阻力沿推力方向与旋转平面投影并沿展向积分，即得 T 与 Q。小迎角下 C_T、C_Q 均随桨距角近似线性增长，随 J 下降——量纲分析中的”常数”实为随工况缓变的系数，集总模型 k_T、k_Q 是其在工作点的局部固化。

3. 反扭矩的角动量守恒推导（严格）

问题：螺旋桨–电机–机体系统中，机体究竟承受多大的反作用扭矩？

取转子（电机转子+桨）与定子（固连机体）为对象。设转子对转轴的转动惯量 J_p，角速度 ω，作用于转子的外力矩有：电机电磁驱动力矩 τ_e（定子→转子，内部作用）与空气对桨的阻力矩 Q（流体→转子，方向阻碍旋转）。转子角动量定理：

J_p·ω̇ = τ_e − Q        ⟹        τ_e = Q + J_p·ω̇

由牛顿第三定律，定子（即机体）承受转子的反作用 −τ_e。因此传给机体的反扭矩等于电磁扭矩而非气动阻力矩：

┌─────────────────────────────────┐
│  τ_机体 = τ_e = k_Q·ω² + J_p·ω̇  │
└─────────────────────────────────┘

物理含义： - 稳态（ω̇ = 0）：机体反扭矩等于气动阻力矩 k_Q·ω²； - 加速：反扭矩被转子惯性放大，机体瞬时承受 Q + J_p·ω̇； - 减速：惯性项反向，反扭矩可小于 Q 甚至反向。

机动过程中的滚转通道扰动放大，根源正在于此。从系统角动量看更本质：转子角动量变化率 J_p·ω̇、排出流体带走的角动量流率 Q、机体角动量变化率三者之和恒为零——三部分在任何时刻精确配平。

4. 电机电磁动态：一阶滞后的来源

永磁无刷电机经换向后可用直流等效：

电学:  u = R·i + L·di/dt + k_e·ω        （k_e·ω 为反电动势）
电磁转矩:  τ_e = k_t·i                   （SI 单位制下 k_t = k_e）
机械:  J_p·ω̇ = τ_e − k_Q·ω²

电时间常数 L/R（毫秒级）通常远小于机械动态，令 di/dt ≈ 0 消去电流：

i = (u − k_e·ω)/R   ⟹   τ_e = (k_t/R)·(u − k_e·ω)
J_p·ω̇ = (k_t·u/R) − (k_t·k_e/R)·ω − k_Q·ω²

在工作点 ω₀ 处线性化（δω̇ 方程）：

J_p·δω̇ = −(k_t·k_e/R + 2·k_Q·ω₀)·δω + (k_t/R)·δu

即一阶惯性环节，时间常数：

τ_m = J_p / (k_t·k_e/R + 2·k_Q·ω₀)

注意它由电磁阻尼 k_t·k_e/R（反电动势反馈）与气动负载斜率 2k_Q·ω₀ 共同构成——螺旋桨的平方负载本身贡献了天然的速度负反馈。仿真中”电机一阶滞后”模型的出处即此。

5. 旋转部件的陀螺力矩（进动方程）

设转子角动量 h = J_p·ω·n（n 为转轴单位矢量，随机体转动）。角动量定理须在惯性系中表述；对随体矢量使用动系中的绝对导数公式：

(dh/dt)|_惯性 = (dh/dt)|_随体 + Ω × h

若转速恒定（随体导数为零），角动量变化完全来自载体转动引起的进动，所需外力矩 M = Ω × h。由作用反作用，转子对机体施加的陀螺反力矩为：

M_陀螺 = −Ω × h = −Ω × (J_p·ω·n)

方向垂直于转轴与机体角速度构成的平面。推论：

  • 单转子飞机横滚时产生俯仰/偏航耦合进动力矩；
  • 等值反向双转子（h₁ = −h₂）使 Ω×(h₁+h₂) = 0，机体角速率引起的陀螺项严格互消；
  • 摆座摆动本身是转轴的附加进动：Ω_摆 = δ̇·e_摆轴，产生 M = −δ̇·J_p·ω·(e_摆轴 × n)，量级 J_p·ω·δ̇。

6. 集总模型总结

推力:      T = k_T·ω²          k_T = C_T·ρ·D⁴/(4π²)
反扭矩:    τ = k_Q·ω² + J_p·ω̇   k_Q = C_Q·ρ·D⁵/(4π²)
电机动态:  τ_m·ω̇ + ω = ω_cmd     τ_m = J_p/(k_t·k_e/R + 2·k_Q·ω₀)
陀螺力矩:  M = −Ω × (J_p·ω·n)
诱导功率:  Pᵢ = T^(3/2)/√(2ρA)   （效率评估基准）

这组方程把”螺旋桨–电机–机体”的相互作用完整封闭：气动负载决定反扭矩与电磁时间常数，电磁驱动决定转速响应，转子角动量决定陀螺耦合——每一个环节都可追溯到流体力学、电磁学或刚体力学的基本定律。
