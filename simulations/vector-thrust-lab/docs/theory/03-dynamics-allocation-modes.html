纯理论推导（三）：刚体动力学、控制分配与稳定性模态分析

本篇是一般性理论：从动量定理推导牛顿–欧拉方程，建立任意多矢量推进器的力/力矩映射与控制分配理论，再对整机动力学做小扰动线性化并导出经典飞行力学模态。不依赖任何具体仿真实现。

1. 牛顿–欧拉方程的推导

1.1 平动部分

刚体可视为质点系。对每个质点 mᵢ 列牛顿第二定律并求和：内力两两抵消（牛顿第三定律），Σmᵢ·aᵢ = M·a_c（质心定义），得质心运动定理：

M·(dv_c/dt)|_惯性 = F_合

在体轴系中表达绝对导数（动系公式）：

(dv/dt)|_惯性 = (dv/dt)|_体轴 + ω × v

于是得到机体系下的平动方程：

┌──────────────────────────────────┐
│  M·v̇ = F_合 − M·(ω × v)           │
└──────────────────────────────────┘

ω×v 一项并非附加假设，而是"在旋转坐标系中表达惯性系加速度"的必然产物（科里奥利项）。

1.2 转动部分

定义刚体对质心的角动量 H = ∫ r×(ω×r) dm = I·ω，其中惯量张量 I 在体轴系中为常矩阵（这正是选用体轴系的动机）。角动量定理 (dH/dt)|_惯性 = M_合，同样展开绝对导数：

M_合 = (dH/dt)|_体轴 + ω × H = I·ω̇ + ω×(I·ω)

即欧拉方程：

┌──────────────────────────────────────────┐
│  I·ω̇ = M_合 − ω×(I·ω)                      │
│  分量形式（主轴系 I = diag(Ix, Iy, Iz)）:    │
│  Ix·ṗ = Mx − (Iz−Iy)·q·r                   │
│  Iy·q̇ = My − (Ix−Iz)·r·p                   │
│  Iz·ṙ = Mz − (Iy−Ix)·p·q                   │
└──────────────────────────────────────────┘

(Iz−Iy)qr 等交叉项是角动量矢量方向随体旋转的几何结果。若刚体含旋转部件（转子角动量 h），总角动量 H = I·ω + h，同一推导给出附加项 −ω×h——陀螺进动耦合自然出现，无需另行假设。

2. 多矢量推进器的一般力/力矩映射

2.1 映射结构

设有 N 个推进器，第 i 个位于 rᵢ，推力轴单位矢量 nᵢ 由摆角 δᵢ 经旋转链给出 nᵢ = Rᵢ(δᵢ)·eₓ，产生推力 Tᵢ 与反扭矩 τᵢ（符号 sᵢ = ±1 表示旋向）。机体系合力/合力矩：

F = Σᵢ Tᵢ·Rᵢ(δᵢ)·eₓ
M = Σᵢ [ rᵢ×(Tᵢ·Rᵢ(δᵢ)·eₓ)  +  sᵢ·τᵢ·Rᵢ(δᵢ)·eₓ ]

注意结构：推力经力臂产生力矩，反扭矩与位置无关、只随摆角投影——两类作用解耦清晰。写成矩阵形式：

┌F┐        ┌ T₁ ┐
│ │ = B(δ)·│ ⋮  │        B(δ) ∈ R⁶ˣᴺ（摆角相关的控制效率矩阵）
└M┘        └ T_N┘

2.2 摆角雅可比

摆角本身也是控制量，效率矩阵对其求导：

∂M/∂δᵢ = rᵢ×(Tᵢ·∂Rᵢ/∂δᵢ·eₓ) + sᵢ·τᵢ·∂Rᵢ/∂δᵢ·eₓ

旋转矩阵对摆角的导数 ∂Rᵢ/∂δᵢ = Rᵢ·[e_摆轴]×（由指数映射 R = exp(δ·[e]×) 求导立得），因此：

∂(Rᵢeₓ)/∂δᵢ = e_摆轴 × (Rᵢeₓ)        ← 几何含义: 推力方向对摆角的灵敏度 = 摆轴×推力轴

整条导数链完全解析，无需求数值差分。

3. 控制分配（Control Allocation）

3.1 问题表述

已知期望广义力 ν_d ∈ Rᵐ（m 个受控力/力矩分量），求执行器向量 u（推力、摆角、差速）使 B(u)·u = ν_d。执行器个数通常大于受控分量（冗余），且受物理约束 u_min ≤ u ≤ u_max。

3.2 线性化分配与伪逆

在工作点将效率矩阵定常化，问题化为欠定线性方程 B·u = ν_d，最小范数解由 Moore–Penrose 伪逆给出：

u* = B⁺·ν_d = Bᵀ·(B·Bᵀ)⁻¹·ν_d

伪逆解的变分意义：min ½‖u‖² s.t. B·u = ν_d（拉格朗日乘子法一步可得）。加权形式 u* = W⁻¹Bᵀ(BW⁻¹Bᵀ)⁻¹ν_d 可按各执行器代价/带宽分配负担——例如让低带宽的差速滚转通道少承担高频成分。

3.3 可达力矩集与奇异性

所有执行器在约束内遍历，ν = B·u 的像构成可达力矩集（凸多面体）。两类退化必须检查：

  1. 权限退化：某通道的可达矩随工况变化。差速反扭滚转通道的权限 ∝ k_Q·ω₀²，随油门降低而平方衰减；
  2. 几何奇异：摆角使推力轴共面/共线时 B 降秩。单轴摆座在 δ = ±90° 处 ∂M/∂δ → 0，分配矩阵条件数发散——控制律设计必须避开或重构。

4. 小扰动线性化（一般框架）

将非线性动力学 ẋ = f(x, u) 在配平点 (x₀, u₀)（满足 f(x₀, u₀) = 0）展开：

δẋ = A·δx + B·δu,        A = ∂f/∂x|₀,  B = ∂f/∂u|₀

配平点的存在性本身是一个非线性方程求解问题（力平衡 + 力矩平衡 + 推力-阻力平衡）。纵向与横航向运动在对称面内解耦（小扰动下），分别导出两组经典模态。

5. 纵向模态的解析推导

5.1 短周期模态

假设 V 近似不变（快变量为 α、q），纵向方程缩减为：

α̇ ≈ (Z_α/V)·α + q          q̇ = M_α·α + M_q·q

特征方程 λ² − (Z_α/V + M_q)·λ + (Z_α/V·M_q − M_α) = 0。二阶标准形对照：

ω_n,sp² ≈ −M_α + (Z_α/V)·M_q        （静稳定度主导）
2·ζ_sp·ω_n,sp ≈ −(Z_α/V + M_q)      （气动阻尼 + 俯仰阻尼主导）

C_mα < 0 给刚度、C_mq < 0 给阻尼——正是这一对导数使"摆角→角加速度"的积分链闭合。

5.2 长周期（Phugoid）模态

慢变量为 V 与航迹角 γ，近似 α 已按短周期平衡。能量观点：动能与势能沿航迹交换。经典近似（Lanchester）：

ω_n,ph ≈ √2·g/V₀          ζ_ph ≈ D/(2·L) 量级（气动比决定阻尼）

长周期阻尼弱、周期长，物理上是"速度↔︎高度"的弱衰减振荡。

6. 横航向模态

模态 近似特征值 物理本质
滚转收敛 λ ≈ L_p < 0 滚转阻尼 C_lp 主导的快衰减
螺旋 λ ≈ (L_β·N_r − L_r·N_β)/（小量） 方向稳定性与上反效应的博弈，可正可负
荷兰滚 ω_n² ≈ N_β，ζ ∝ −N_r 航向静稳定给刚度、偏航阻尼给衰减的侧滑-滚转耦合振荡

三者同样全部由"静稳定导数给刚度、速率导数给阻尼"的同一逻辑支配——这与推进矢量通道的稳定性结构完全同构。

7. 解耦设计的理论条件

对多通道矢量推进布局，通道近似解耦要求效率矩阵的交叉项满足：

|∂M_i/∂u_j| / |∂M_i/∂u_i| ≪ 1      (i ≠ j)

对"前后双发、正交摆轴、差速滚转"这一类布局，交叉项量级为 τ/(T·力臂) ∝ D·(C_Q/C_T)/力臂——即反扭-推力比与力臂之比决定耦合强度，与具体参数无关的结构性强结论。当该比值足够小，三通道可按 SISO 分别整定，残余耦合以前馈补偿；反之则必须按 MIMO 系统设计。
