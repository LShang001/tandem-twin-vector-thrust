MOD-003 配平与稳定性分析

定位：在 MOD-001 整机动力学推导基础上，定量核对纵列双发矢量推力飞行器的纵向配平条件，分析零迎角俯仰力矩 C_m0 的补偿需求，并给出小扰动稳定性框架与数值证据。本文档为概念级分析，参数均取自 models/aircraft-model.json（MODEL-DEFAULT）。

1. 配平问题定义

先考察纵向定直平飞参考状态（steady level flight reference）：

V  = 24 m/s        （配平速度）
γ  = 0             （航迹倾斜角为零，水平飞行）
β  = 0             （无侧滑）
φ  = 0, p = q = r = 0   （无滚转、无角速度）
δ_f = 0, Δω = 0    （前摆/差速为零）

此处只联立纵向力与俯仰力矩：

L ≈ m·g            （升力平衡重力）
T ≈ D              （推力平衡阻力）
M_y = 0            （俯仰力矩为零）

代码中配平初始条件由 state.mjs:resetFlightState 设置：

F.vel.x = P.vTrim * cos(P.aTrim);
F.vel.z = P.vTrim * sin(P.aTrim);
S.quat  = (0, sin(P.aTrim/2), 0, cos(P.aTrim/2));

其中 P.vTrim = 24 m/s、P.aTrim = 0.028398719 rad（1.627°）、P.thrTrim = 0.44135018，尾摆配平偏置 P.dtTrim = -0.011344527 rad（-0.650°）。按照项目锁定的欧拉角提取约定，物理俯仰为 +aTrim 时 HUD 显示 theta = -aTrim；二者不能混用。

2. 纵向配平求解

2.1 联立平衡方程

在 V = 24 m/s、beta = p = q = r = delta_f = Delta omega = 0 且航迹角为零时，物理俯仰角等于迎角 alpha。令双发稳态推力相等为 T = k_T(thr*w_max)^2，未知量取 [alpha, thr, delta_t]。将推进力、线性气动力和重力全部写入机体系，纵向三方程平衡条件为：

0 = T(1+cos(delta_t)) + L*sin(alpha) - D*cos(alpha) - m*g*sin(alpha)
0 = -T*sin(delta_t) - L*cos(alpha) - D*sin(alpha) + m*g*cos(alpha)
0 = -b*T*sin(delta_t) + qbar*S*cbar*(Cm0 + Cma*alpha)

其中 qbar = rho*V^2/2、L = qbar*S*CLa*alpha、D = qbar*S*(CD0 + CDk*(CLa*alpha)^2)。该三方程同时约束纵向力、法向力和俯仰矩，不能用单独的 L=W 与 T=D 代替，因为尾摆同时引入垂向推力和俯仰力矩。

2.2 数值解与代码参数

使用 models/aircraft-model.json 中的 MODEL-DEFAULT 参数求解得到：

量 数值 代码参数
配平迎角 alpha_0 0.028398719 rad = 1.62713° aTrim
配平油门 0.44135018 thrTrim
尾摆配平偏置 delta_t0 -0.011344527 rad = -0.64999° dtTrim
单发转速 397.215 rad/s thrTrim*wMax
单发推力 3.47116 N kT*(thrTrim*wMax)^2

代回上述三式后，F_x、F_z、M_y 三个残差均小于 6e-11（SI 单位）。这不是六自由度全配平：非零 dtTrim 使尾电机反扭矩在 z_b 轴留下 M_z=-Q_t sin(dtTrim)≈+0.00286 N·m 的一阶残差。当前没有联立求解前摆、差速、侧滑和滚转姿态来同时闭合 F_y、M_x、M_z。

在 60 Hz 帧率、4 ms 子步和全 SAS 模式下连续运行 10 s，回归基线最终空速为 24.000085 m/s、高度漂移 0.000244 m、显示俯仰角 -1.62696°。同时出现 phi=-0.05229°、psi=-0.16778° 和侧移 y=0.27497 m。因此该初值应称为“纵向配平参考点”；它适合验证纵向平衡与控制符号，但不能直接作为完整九状态 Jacobian 的平衡点。

2.3 指令零位与显示角

UI 中尾摆滑块表示相对配平增量，实际执行角为 delta_t,act = dtTrim + delta_t,UI + delta_t,SAS。因此滑块为零不代表物理尾摆角为零，而代表处于 dtTrim 配平偏置。全 SAS 的俯仰姿态误差定义为：

theta_error = theta_display - theta_ref = theta_display + aTrim
theta_ref = -aTrim

保持这一显式定义可以避免将物理俯仰 +aTrim、HUD 显示 -aTrim 和控制器误差混为一谈。

3. C_m0 案例：零迎角俯仰力矩的配平补偿

3.1 问题来源

气动模型在零迎角处存在非零俯仰力矩：

C_m0 = 0.011

其物理来源为机翼弯度/安装角；在零迎角时产生的气动力矩为：

M_y,a|_{α=0} = q̄·S_w·c̄·C_m0
             = 352.8 × 0.55 × 0.35 × 0.011
             ≈ 0.747 N·m

该力矩方向为正（机头抬头，绕 +y_b）。若不用尾摆角补偿，飞行器将绕 y_b 持续抬头，迎角增大；由于 C_mα = −0.4 < 0（静稳定），最终会在某个正迎角处由 C_m0 + C_mα·α = 0 平衡，但此时需要推力矢量或其他控制量维持。

3.2 用尾摆角平衡 C_m0 的需求

尾电机推力对质心的俯仰力矩（小摆角线性化）为：

M_y,t = −b · T_t · sin(δ_t) ≈ −b · T_t · δ_t

其中 b = 0.62 m，T_t = 3.471 N（当前 thrTrim）。令总俯仰力矩为零：

−b·T_t·sin(δ_t) + q̄·S_w·c̄·C_m0 = 0

解得：

sin(δ_t,trim) = (q̄·S_w·c̄·C_m0) / (b·T_t)
              = 0.747 / (0.62 × 3.471)
              ≈ 0.347
δ_t,trim ≈ asin(0.347) ≈ 0.354 rad ≈ 20.3°

3.3 执行机构裕度

摆角限幅：

δ_max = P.dMax = 0.436 rad = 25°

所需配平摆角占限幅比例：

20.3° / 25° ≈ 81 %

剩余裕度仅约 19%。因此，在零迎角情况下理论上仍可仅靠尾电机摆角补偿 C_m0，但会占用大部分俯仰控制权限，且还没有计入扰动和模型不确定性。

3.4 与 aTrim 的耦合

代码配平迎角 aTrim = 1.627° 接近气动自平衡点：

α_c = −C_m0 / C_mα = −0.011 / (−0.4) = 0.0275 rad = 1.576°

在该迎角下：

C_m0 + C_mα·aTrim ≈ 0.011 − 0.4 × 0.028399 ≈ −3.60×10⁻⁴

对应气动俯仰矩约为 -0.0244 N·m，由 dtTrim=-0.650° 产生的正向尾摆力矩精确平衡。SAS 积分器只处理模型误差和扰动，不承担静态配平职责。

4. 小扰动线性化框架

4.1 状态与输入定义

若先求得六自由度全配平，可选取机体坐标系小扰动状态：

x = [ Δu, Δw, Δq, Δθ, Δv, Δp, Δr, Δφ, Δψ ]^T

输入为执行机构修正量：

u = [ Δδ_t, Δδ_f, ΔΔω ]^T

其中：

  • 纵向通道：Δu, Δw, Δq, Δθ；输入 Δδ_t。
  • 横侧向通道：Δv, Δp, Δr, Δφ, Δψ；输入 Δδ_f（偏航）、ΔΔω（滚转）。

4.2 纵向模态定性分析

纵向力与力矩在配平点附近线性化：

X = X_u Δu + X_w Δw + X_δt Δδ_t
Z = Z_u Δu + Z_w Δw + Z_δt Δδ_t
M = M_u Δu + M_w Δw + M_q Δq + M_δt Δδ_t

典型固定翼纵向模态：

模态 主导状态 稳定条件 本项目依据
短周期 Δα（或 Δw）、Δq M_α < 0（静稳定）、M_q < 0（阻尼） C_mα = −0.4 < 0、C_mq = −9.0 < 0，静稳定且阻尼为负
长周期 / 沉浮 Δu、` Δθ` 推力 − 阻力曲线在配平点附近斜率为负

结论：导数符号支持短周期静稳定与角速率阻尼，10 s 纵向参考量保持良好；横航向仍存在可测残差。这既不能替代特征值、参与因子和更长时域验证，也不能把当前纵向工作点直接称为完整线性化平衡点。

4.3 横侧向模态定性分析

横侧向力与力矩线性化：

Y = Y_v Δv + Y_δf Δδ_f
L = L_v Δv + L_p Δp + L_δω ΔΔω
N = N_v Δv + N_r Δr + N_δf Δδ_f

典型固定翼横侧向模态：

模态 主导状态 稳定条件 本项目依据
荷兰滚 Δβ、Δr、Δφ 耦合振荡 C_nβ > 0（航向静稳定）、C_nr < 0（偏航阻尼） C_nβ = 0.12 > 0、C_nr = −0.2 < 0，航向稳定且有阻尼
滚转收敛 Δp C_lp < 0（滚转阻尼） C_lp = −0.5 < 0，滚转运动快速收敛
螺旋 Δφ 缓慢收敛/发散 C_lβ < 0（横滚静稳定）抑制发散 C_lβ = −0.09 < 0，有利于螺旋模态稳定

滚转控制特殊性：本布局滚转由双发反扭矩差驱动，控制效率 ∂M_x/∂Δω ≈ −2·k_Q·ω₀² 为负，SAS 符号已按此整定；低油门时反扭矩差基数减小，滚转权限衰减。

4.4 适用边界

上述线性化仅在以下条件有效：

  • 小迎角、小侧滑角、小摆角；
  • 气动模型为线性，未计入失速、压缩性、螺旋桨滑流干扰；
  • 推力/阻力平衡近似，长周期分析需配合精确油门；
  • 无风、理想执行器（无舵机速率饱和、无安装误差）。

5. 数值证据

回归基线定义于 tests/scenarios.mjs，采用固定 60 Hz 帧步长驱动 stepPhysics，覆盖三条工况：

工况 描述 观察指标 结论
A：纵向配平参考 10 s 无额外输入，从纵向三方程平衡状态出发 V=24.000085 m/s、z=0.000244 m、theta=-1.62696°；phi=-0.05229°、psi=-0.16778°、y=0.27497 m 纵向三残差闭合；横航向残差有界但非零
B：俯仰增量阶跃 8° / 4 s 从 NED z=-100 m 开始，delta_t,UI = 8° 持续 240 帧后撤除 8 s 时 V=25.4292 m/s、theta=-2.9525°、q=0.023789 rad/s，全程未触地 验证自由飞行响应与撤除事件；8 s 内尚未完全回到参考点
C：滚转阶跃 0.25·dwUiMax / 4 s Delta omega = 0.1375 持续 240 帧后撤除 8 s 时 phi=0.5284°、p=0.002376 rad/s 姿态误差显著衰减；仍非全包线稳定性证明

上述结果属于演示性数值检查，仅在所列工况、MODEL-DEFAULT 参数、当前线性气动模型与理想执行器假设下观察到响应有界。不能外推为大机动、失速、悬停或全包线稳定结论。详细验证流程与通过准则见 VER-001。

6. 已知限制与未覆盖项

  1. 线性气动：未含大迎角失速、非定常气动力、螺旋桨滑流与机身干扰。
  2. 欧拉角奇异：eulerFromQuat 在 theta = ±90° 附近出现万向锁；代码未做特殊处理，大俯仰机动可能数值不稳定。
  3. 低速 / 悬停：真实空速为零时动压与气动力严格为零，姿态稳定完全依赖推进控制；lockXY 只是外部运动学约束，不构成悬停能力证明。
  4. 理想执行器：电机为一阶滞后、摆角瞬时到达指令、无舵机带宽与饱和特性。
  5. 无风环境：未引入大气扰动、阵风或风切变。
  6. 仅纵向配平：当前参数只闭合 F_x、F_z、M_y；完整稳定性矩阵前必须继续求解横航向平衡。
