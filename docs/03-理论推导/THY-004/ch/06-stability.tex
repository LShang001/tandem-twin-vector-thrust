\section{线性化与稳定性模态分析}
\label{sec:stability}

线性化可用于分析配平点附近的局部稳定性，但其结论依赖配平状态、气动导数、推进工作点和控制增益。
本项目参数均为未经标定的MODEL-DEFAULT；仓库当前尚未形成经复核的数值Jacobian与特征值数据集。
因此本节给出可复现的方法、经典模态的物理解释和反馈增益的定性作用，不报告虚构的极点坐标、周期或阻尼比。

\subsection{局部线性化方法与符号边界}

设非线性模型为$\dot{\bm{x}}=\bm{f}(\bm{x},\bm{u})$。完整状态线性化要求参考点满足
$\bm{f}(\bm{x}_0,\bm{u}_0)=\bm{0}$；在该点附近定义扰动量，则
\begin{equation}
    \Delta\dot{\bm{x}}
    = \underbrace{\left.\frac{\partial\bm{f}}{\partial\bm{x}}\right|_0}_{\bm{A}}
      \Delta\bm{x}
    + \underbrace{\left.\frac{\partial\bm{f}}{\partial\bm{u}}\right|_0}_{\bm{B}}
      \Delta\bm{u} .
    \label{eq:linear_state_space}
\end{equation}
在常规定直平飞且纵横向耦合较弱时，可用
\begin{align}
    \bm{x}_{\mathrm{long}} &= [\Delta u,\Delta w,\Delta q,\Delta\theta]^T,\\
    \bm{x}_{\mathrm{lat}}  &= [\Delta v,\Delta p,\Delta r,\Delta\phi]^T
\end{align}
辅助解释经典模态；这只是局部分块近似，并不意味着完整六自由度模型严格解耦。

数值线性化必须直接调用\texttt{src/core/}中的同一状态更新函数，并保持项目约定：NED右手系、
四元数显式欧拉右乘积分以及$\theta=-\arcsin(R_{13})$。尤其不能把其他教材的欧拉角符号矩阵
未经变换直接套到当前显示角变量上。有限差分步长应做至少一个数量级的敏感性检查，
并用非线性小扰动时域响应验证$\bm{A}$、$\bm{B}$的符号和局部预测能力。

\subsection{经典模态的物理解释}

对具有常规静稳定与阻尼特性的固定翼，纵向响应通常可分为：
\begin{itemize}
    \item \textbf{短周期模态}：主要由迎角（或$w$）与俯仰速率$q$耦合，受俯仰静稳定导数和俯仰阻尼导数支配；
    \item \textbf{长周期模态}：主要表现为速度、姿态与高度之间较慢的动能--势能交换，受升阻特性和推力平衡显著影响。
\end{itemize}
横航向响应通常可分为：
\begin{itemize}
    \item \textbf{滚转收敛模态}：以滚转速率$p$为主，气动滚转阻尼决定其主要衰减趋势；
    \item \textbf{荷兰滚模态}：侧滑、偏航速率与滚转运动耦合形成的振荡，其频率与阻尼由多个横航向导数共同决定；
    \item \textbf{螺旋模态}：滚转角、侧滑与偏航的慢耦合，可能缓慢收敛或发散，符号不能脱离具体导数判断。
\end{itemize}
这些名称来自经典飞行动力学\cite{etkin1982dynamics,stengel2004flight}，但当前C0模型是否呈现清晰的
时间尺度分离，仍须由数值特征值、模态参与因子和时域响应共同确认。

\subsection{SAS增益的定性作用}

在线性反馈写成$\Delta\bm{u}=\bm{K}\Delta\bm{x}$时，闭环矩阵为
\begin{equation}
    \bm{A}_{\mathrm{cl}}=\bm{A}+\bm{B}\bm{K}.
\end{equation}
在执行器未饱和、推力非零且反馈符号正确的局部条件下，可作如下定性判断：
\begin{itemize}
    \item $k_q$通过尾摆主通道增加等效俯仰速率阻尼，$k_\theta$增加姿态恢复作用，因而主要影响短周期响应；
    \item $k_r$通过前摆主通道增加等效偏航速率阻尼，通常首先影响荷兰滚响应；
    \item $k_p$通过差速反扭通道增加等效滚转速率阻尼，$k_\phi$增加滚转姿态恢复作用，主要影响滚转收敛；
    \item 长周期与螺旋模态受多项气动和能量耦合控制，内环增益对它们的影响较弱且可能随工作点改变，不能预先断言极点移动方向。
\end{itemize}
以上判断还受电机一阶滞后、摆角与差速限幅、反扭矩交叉项以及低油门效能退化影响。

\subsection{Web core非线性回归证据}

为检查上述符号判断是否与可执行模型一致，图\ref{fig:web_validation}直接使用Web core生成的
基线回归样本。仿真以
$1/60\ \mathrm{s}$帧步长运行，物理子步上限为$0.004\ \mathrm{s}$；配平场景从
$V_0=24\ \mathrm{m/s}$、$\theta_0=-1.627^\circ$和尾摆偏置
$\delta_{t0}=-0.650^\circ$启动。俯仰场景施加$8^\circ$尾摆增量，滚转场景施加
$0.25\,\Delta\omega_{\mathrm{UI,max}}=0.1375$差速指令，均在$t=4\ \mathrm{s}$撤除。
俯仰场景从NED坐标$z=-100\ \mathrm{m}$开始，以排除地面钳位对自由飞行响应的污染。

当前参数只联立闭合$F_x$、$F_z$和$M_y$，属于纵向三方程配平，而不是六自由度全配平。
十秒样本的最大空速偏差小于$0.09\ \mathrm{mm/s}$，高度漂移为$0.244\ \mathrm{mm}$；
尾摆反扭投影留下的横航向残差则导致$0.052^\circ$滚转漂移、$0.168^\circ$航向漂移和
$0.275\ \mathrm{m}$侧移。因此该工作点可用于纵向局部检查，但在形成完整数值Jacobian前，
还需求解横航向平衡。两个阶跃响应同时显示了执行通道方向、撤除后的恢复趋势以及
尚未在八秒窗口内完全回零的闭环行为。它们属于非线性确定性回归证据，不能据此给出
特征值、阻尼比、稳定裕度或飞行品质等级。

\subsection{形成可验证结论所需的证据}

后续稳定性报告应至少包含：配平状态及残差、有限差分设置、连续与离散Jacobian、开闭环特征值、
左右特征向量或参与因子、步长敏感性，以及与非线性小扰动响应的对照。只有在这些证据齐全后，
才应报告具体模态频率、阻尼比或“稳定性显著改善”等定量结论。
