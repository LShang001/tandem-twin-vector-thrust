# GOV-001 项目范围与术语

> 本篇界定「纵列双发矢量推力飞行器」整机工程项目的治理边界：说明项目定位、目标、包含与不包含的工作范围，罗列当前资产与文档体系，并给出术语与缩略语统一表。本篇不展开技术推导，具体物理与数学内容见 THY/MOD/CTL 系列文档。

---

## 1. 项目定位与目标

本项目是一个**概念级六自由度（6-DOF）飞行器仿真平台**，研究对象为纵列双发布局、矢量推力驱动的固定翼/类多旋翼混合构型：

- **前电机**：位于机头，拉力式，正转（CW），绕机体系 z 轴单轴摆动，承担偏航主控；
- **尾电机**：位于机尾，推进式，反转（CCW），绕机体系 y 轴单轴摆动，承担俯仰主控；
- **滚转**：由前后电机转速差速（反扭矩差）控制。

零摆角时前后推力线均通过质心，物理力臂 `a = b = 0.62 m`（注意：渲染几何中电机位置约 `±1.78 m`，仅用于可视化，不进入动力学计算）。

项目目标：

1. 建立可复现、可追踪的 6-DOF 数学模型与控制律研究平台；
2. 提供基于 Web 的交互式仿真演示载体，用于教学、方案迭代与概念验证；
3. 实践「参数单一事实源 + 生成代码 + 自动化回归测试」的轻量级研发治理流程。

项目成熟度声明：**概念级仿真**。所有参数当前均为 `MODEL-DEFAULT`，未经过台架标定或飞行辨识；模型仅用于控制律结构研究与可视化演示，不具备高保真飞行预测能力。

## 2. 范围

### 2.1 包含范围（In-Scope）

| 领域 | 内容 |
|------|------|
| 仿真实现 | Web 六自由度仿真 `simulations/vector-thrust-lab`（Three.js 无构建原生 ESM） |
| 理论推导 | 旋转与姿态数学、推进系统物理、刚体动力学与控制分配（见 THY-001~003） |
| 数学建模 | 6-DOF 仿真模型规范、坐标系与符号约定、配平与稳定性分析（见 MOD-001~003） |
| 参数治理 | `models/aircraft-model.json` 单一事实源、参数同步工具 `tools/sync-params.py` |
| 验证证据 | Node.js 内置测试 runner、40 项断言、3 条回归基线（配平直飞 / 俯仰阶跃 / 滚转阶跃） |
| 文档体系 | 本清单所列全部治理/设计/需求/理论/建模/控制/推进/验证/决策/参考资料文档 |

### 2.2 不包含范围（Out-of-Scope）

| 领域 | 说明 |
|------|------|
| 飞行硬件 | 不设计、不制造、不采购机体结构、电机、电调、传感器或飞控计算机 |
| 适航与认证 | 不涉及民航/军品适航审定、安全关键软件开发流程（如 DO-178C） |
| 实物制造 | 不含机加、复材、总装、台架或外场试验；若未来扩展，应另立项并更新范围 |
| 高保真气动 | 当前采用线性导数模型，不含 CFD、风洞或涡格法气动数据库 |

## 5. 术语与缩略语表

| 术语/缩略语 | 说明 |
|-------------|------|
| 6-DOF | 六自由度（Three-Degrees-of-Freedom Translation + Three-Degrees-of-Freedom Rotation） |
| NED | 北东地坐标系（North-East-Down），惯性/世界参考系，x 指北、y 指东、z 指地 |
| 机体系 {b} | 机体固连坐标系，原点在质心 CG，具体轴向约定见 MOD-002 |
| CG / 质心 | Center of Gravity，飞行器质量中心，机体系原点 |
| DCM | Direction Cosine Matrix，方向余弦矩阵，即 SO(3) 旋转矩阵 R |
| 四元数 | 单位四元数 q = (q0, q1, q2, q3)，用于无奇点姿态积分；本项目采用右乘机体系角速度形式 |
| 欧拉角 | 航空 3-2-1 顺序（偏航 ψ / 俯仰 θ / 滚转 φ），仅用于显示与控制反馈提取 |
| α（迎角） | Angle of Attack，机体系 x 轴与空速矢量投影夹角 |
| β（侧滑角） | Sideslip Angle，空速矢量相对机体对称面的侧向偏角 |
| q̄（动压） | 动压 `q̄ = 0.5·ρ·V²`，气动升/阻/力矩计算的基础 |
| 矢量推力 | 通过电机整体摆动改变推力方向，产生直接力与控制力矩 |
| δf / δt | 前/后电机摆角（前 = front，尾 = tail），机体系 y/z 轴旋转 |
| Δω / 差速 | 前后电机转速相对配平的归一化差值，滚转控制通道 |
| 配平（Trim） | 使机体在稳态飞行中合外力与合外力矩为零的状态/参数组合 |
| 反扭矩 | 螺旋桨旋转产生的绕机体纵轴反作用力矩，双发反向旋转以部分抵消 |
| 拉力式 | 螺旋桨位于机翼/机头前方，拉力驱动气流过翼面 |
| 推进式 | 螺旋桨位于机身后方，推动机体前进 |
| SAS | Stability Augmentation System，增稳系统；本项目为速率+姿态比例+积分反馈 |
| MODEL-DEFAULT | 参数状态 6 级之一：模型默认值，无标定/验证证据 |
| ASSUMED | 参数状态 6 级之一：基于工程假设的估计值 |
| ESTIMATED | 参数状态 6 级之一：基于分析/类比计算得到的估计值 |
| DATASHEET | 参数状态 6 级之一：来自公开或厂商数据手册 |
| BENCH-TESTED | 参数状态 6 级之一：经过地面台架测试标定 |
| FLIGHT-IDENTIFIED | 参数状态 6 级之一：经过飞行试验辨识 |
| 回归基线 | `tests/fixtures/regression-baseline.json` 记录的标准工况时序输出，容差 1e-9 |
| 行为保持红线 | 仿真实现中不可随意改动的数值/算法契约，修改需重采基线并记录原因 |
