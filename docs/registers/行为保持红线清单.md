# 行为保持红线清单

> 修改以下任一项会改变仿真可观测行为。所有有意变更必须：
> 1. 更新本文档中的对应描述
> 2. 重采回归基线 `simulations/vector-thrust-lab/tests/fixtures/regression-baseline.json`
> 3. 在提交说明中明确记录变更原因与影响范围

## 红线项

| 红线项 | 当前值 | 所在位置 | 说明 |
|--------|--------|----------|------|
| 帧 delta 上限 | 0.05 s | `frameCap` | 主循环最大时间步长 |
| 积分子步上限 | 0.004 s | `maxStep` | 显式欧拉稳定性要求 |
| 电机一阶滞后时间常数 | 0.28 s | `tauM` | 电机+桨负载综合响应 |
| 差速开方分配 | `ωf=ω0√(1+Δω)`, `ωt=ω0√(1−Δω)` | `propulsion.mjs` | 保总推力不变的平方和约束 |
| 推力公式 | `T = kT·ω²` | `propulsion.mjs` | 平方关系，非其它指数 |
| 反扭矩公式 | `τ = kQ·ω² + Jp·dω/dt` | `propulsion.mjs` | 含惯性项 |
| 六维力映射 | 见 `propulsion.mjs:28-34` | `propulsion.mjs` | Fx/Fz/My 符号约定与推导已交叉验证 |
| SAS 摆角限幅 | ±0.4363 rad (25°) | `dMax` | 物理摆座行程限制 |
| SAS 差速限幅 | ±0.7 | `dwMax` | 归一化差速范围 |
| SAS 积分限幅 | 俯仰 ±0.5 rad / 滚转 ±0.3 rad | `intThMax`/`intPhiMax` | 防积分饱和 |
| UI 差速幅值 | ±0.55 | `dwUiMax` | 用户界面滑块范围 |
| 四元数积分 | 显式欧拉右乘 + 每步归一化 | `dynamics.mjs` | `q̇ = ½·q⊗ω` |
| 欧拉角提取 | `theta = -asin(R13)` | `math.mjs` | NED 3-2-1 序列 |
| 空速分母保护 | 0.5 m/s | `vMin` | 仅保护侧滑角与无量纲角速率分母；动压使用真实空速 |
| 地面约束 | `pos.z > 6.2` 且 `vWorld.z > 0` | `groundZ` + `state.mjs` | 仅拦截下落，不含地面摩擦 |

| 自动综合演示 | 900 ms 触发 | `demo.mjs` | 超时自动启动演示序列 |
| 完整复位行为 | 恢复配平油门/转速/指令/执行量/时间并清零遥测 | `resetSimulationState` | 保留 SAS、气动和水平约束模式开关 |
| 配平工作点 | `V=24 m/s`, `alpha=1.62713°`, `thr=0.441350`, `delta_t0=-0.64999°` | `aTrim`/`vTrim`/`thrTrim`/`dtTrim` | 三个纵向平衡残差同时闭合 |
| SAS 姿态运动学 | `theta_dot≈-q`, `phi_dot≈-p` | `control.mjs` | 速率项取正，显示姿态 P/I 项取负 |

## 已验证的约定（非红线，但修改需注意）

| 约定 | 详情 | 交叉验证来源 |
|------|------|-------------|
| 尾电机受力方向 | `[+cosδt, 0, -sinδt]`（牛顿第三定律：与排气反向） | `propulsion.mjs:28-34` |
| 尾电机角动量方向 | `[-cosδt, 0, +sinδt]`（CCW 反转，h_t 沿 -u_F,t） | `dynamics.mjs:48-52` |
| 前后角动量对消 | `hv.x = Jp(wf·cf - wt·ct)` | `dynamics.mjs:49` |
| 耦合比 | `(C_Q/C_T)·(D/b)·(δf/δt)`，典型 3%-15% | 量纲分析 + aircraft-model.json |
| NED M_y 约定 | M_y > 0 = 机头向上，M_y < 0 = 机头向下（+y_b 右手定则） | `dynamics.mjs` 角加速度与 `eulerFromQuat()` R₁₃/θ 推导 |
