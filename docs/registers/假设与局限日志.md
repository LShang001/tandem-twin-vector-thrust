# 假设与局限日志

> 本日志记录「纵列双发矢量推力飞行器」C0 仿真模型中的所有已知假设、简化与局限。每项均说明其影响域、对结果可信度的影响评估、解除条件或改进方向，以及关联文档。新增假设应经变更评审后按顺序编号加入。

---

|------|-------------|--------|----------|----------------------|----------|
| ASM-001 | **点质量刚体**：仅考虑质心平动与绕质心转动，无弹性变形、无附加质量、无惯量积 | 动力学、配平、模态分析 | 概念级分析可接受；高机动或柔性机体时会低估耦合与振动 | 引入结构有限元或多体模型，通过台架 / 模态试验更新惯量张量 | [MOD-001](../04-数学建模/MOD-001-六自由度仿真模型规范.md)、`src/core/dynamics.mjs` |
| ASM-002 | **理想执行机构**：前 / 尾摆角与差速指令瞬时到位，无舵机速率 / 带宽 / 间隙模型 | 控制、推进、频域分析 | SAS 增益按理想执行机构整定，真实系统中高频相位损失可能导致振荡 | 增加执行机构一阶滞后或速率饱和模型；当前 REQ-CTL-003 列为未满足 | [CTL-001](../05-控制与分配/CTL-001-控制架构与SAS.md)、[REQ-001](../02-需求与验证/REQ-001-需求基线.md) REQ-CTL-003 |

| ASM-006 | **推力线过质心**：零摆角时前后电机推力线均通过 CG，无静态推力力矩 | 推进、配平、稳定性 | 简化了配平计算；真实安装误差会产生附加力矩 | 测量真实推力线相对 CG 的偏移并引入安装矩阵 | [CFG-000](../01-方案设计/CFG-000-概念构型基线C0.md)、`src/core/propulsion.mjs` |
| ASM-007 | **旋翼陀螺效应简化**：仅保留 `Jp·ω̇` 与进动项 `ω × h_转子`，未完整建模陀螺力矩 / 挥舞 / 锥进 | 转动动力学、高机动响应 | 中低速小机动下可接受；快速滚转 / 俯仰组合机动时耦合精度有限 | 建立完整旋翼动力学或多桨叶坐标模型 | [THY-002](../03-理论推导/THY-002-推进系统的物理建模.md)、`src/core/dynamics.mjs` |
| ASM-008 | **渲染非尺度化**：三维模型几何臂长 `±1.78 m` 为示意性可视化尺寸，不等于物理力臂 `a = b = 0.62 m` | UI / 可视化 | 不影响物理计算，但可能误导观众对尺寸与力臂的直观判断 | 文档化渲染尺度；可选提供按真实尺寸绘制的辅助模型 | [MOD-002](../04-数学建模/MOD-002-坐标系与符号约定.md)、README |

| ASM-010 | **欧拉 ±90° 奇异未处理**：采用 3-2-1 欧拉角提取，俯仰角接近 ±90° 时出现万向锁 | 显示、SAS、HUD | 大俯仰机动时欧拉角不连续，SAS 可能基于错误姿态角输出 | 限制演示机动范围；或在控制律中改用四元数直接反馈 | [THY-001](../03-理论推导/THY-001-旋转与姿态的数学基础.md)、`src/core/math.mjs` |
| ASM-011 | **无传感器模型**：角速度、姿态、位置、速度均直接可用，无噪声、无延迟、无失效 | 控制、导航、估计 | 无法验证状态估计算法、滤波器及传感器失效保护 | 增加 IMU、GPS、空速管模型及相应滤波器 | [CTL-001](../05-控制与分配/CTL-001-控制架构与SAS.md) |
| ASM-012 | **无电调 / 电池动态**：电机响应仅由一阶滞后 `tauM` 描述，忽略电流环、电压跌落、热衰减 | 推进、能源管理 | 瞬态推力与真实电调-电池系统存在差异 | 增加电调电流环、电池内阻与 SOC 热模型 | [PROP-001](../06-推进与执行机构/PROP-001-推进与摆座模型.md) |

## 使用说明

- 任何人在使用仿真结果时，应首先查阅本日志确认目标工况是否落在模型适用范围内。
- 若因新增功能、试验数据或设计变更导致某条假设不再成立，应更新本日志状态并新增替代假设或解除记录。
- 本日志与 [`危害与失效模式日志`](./危害与失效模式日志.md)、[SAFE-001](../07-验证与确认/SAFE-001-初步安全计划.md) 共同构成模型成熟度声明的支撑文件。
