# MOD-003 配平与稳定性分析

> 定位：在 MOD-001 整机动力学推导基础上，定量核对纵列双发矢量推力飞行器的纵向配平条件，分析零迎角俯仰力矩 `C_m0` 的补偿需求，并给出小扰动稳定性框架与数值证据。本文档为概念级分析，参数均取自 `models/aircraft-model.json`（MODEL-DEFAULT）。

---

## 1. 配平问题定义

考察**定直平飞**（steady level flight）状态：

```
V  = 24 m/s        （配平速度）
γ  = 0             （航迹倾斜角为零，水平飞行）
β  = 0             （无侧滑）
φ  = 0, p = q = r = 0   （无滚转、无角速度）
δ_f = 0, Δω = 0    （前摆/差速为零）
```

此时纵向力与力矩需满足：

```
L ≈ m·g            （升力平衡重力）
T ≈ D              （推力平衡阻力）
M_y = 0            （俯仰力矩为零）
```

代码中配平初始条件由 `state.mjs:resetFlightState` 设置：

```javascript
F.vel.x = P.vTrim * cos(P.aTrim);
F.vel.z = P.vTrim * sin(P.aTrim);
S.quat  = (0, sin(P.aTrim/2), 0, cos(P.aTrim/2));
```

其中 `P.vTrim = 24 m/s`、`P.aTrim = 0.027925268 rad`（1.6°）、`P.thrTrim = 0.5`。

## 2. 纵向配平推导

### 2.1 升力平衡 → 迎角

重力：

```
W = m·g = 2.6 kg × 9.81 m/s² = 25.506 N
```

动压：

```
q̄ = ½·ρ·V² = 0.5 × 1.225 kg/m³ × (24 m/s)² = 352.8 Pa
```

升力面积项：

```
q̄·S_w = 352.8 Pa × 0.55 m² = 194.04 N
```

由 `L = q̄·S_w·C_L` 及平飞 `L = W`，得配平升力系数：

```
C_L,trim = W / (q̄·S_w) = 25.506 / 194.04 ≈ 0.1314
```

本仿真采用线性升力模型 `C_L = C_Lα·α`，故：

```
α_trim, Lift = C_L,trim / C_Lα = 0.1314 / 4.6 ≈ 0.02857 rad ≈ 1.638°
```

### 2.2 与代码给定 aTrim 的核对

代码给定：

```
P.aTrim = 0.027925268031909273 rad = 1.600°
```

两者偏差：

```
Δα = 0.027925 − 0.02857 = −0.00065 rad ≈ −0.038°
相对偏差 ≈ −2.3 %
```

结论：`P.aTrim` 是将理论升力平衡迎角按 `1.6°` 圆整后的结果，偏差在 3% 以内，可视为近似配平。剩余升力残差由 SAS 俯仰积分器在仿真中逐步吸收。

### 2.3 阻力与推力平衡 → 油门

在 `α_trim = 1.6°` 处：

```
C_L,trim = C_Lα · P.aTrim = 4.6 × 0.027925 ≈ 0.1285
```

阻力系数：

```
C_D = C_D0 + C_Dk · C_L²
    = 0.035 + 0.045 × (0.1285)²
    ≈ 0.035 + 0.000743
    ≈ 0.03574
```

平飞所需总推力：

```
D = q̄·S_w·C_D = 194.04 N × 0.03574 ≈ 6.94 N
```

代码配平油门 `P.thrTrim = 0.5` 对应的电机转速：

```
ω₀ = thrTrim · ω_max = 0.5 × 900 rad/s = 450 rad/s
```

单发推力：

```
T = k_T · ω₀² = 2.2×10⁻⁵ N·s² × (450 rad/s)² = 4.455 N
```

双发总推力（Δω = 0）：

```
T_total = 2 × 4.455 N = 8.91 N
```

推力 − 阻力：

```
T_total − D = 8.91 N − 6.94 N = 1.97 N
```

即在该配平点，推力约比阻力大 28%。多余推力将使飞行器加速，直至阻力随速度平方增长而重新平衡。MOD-001 的离线复算观察到配平后速度上升至约 26.2 m/s，与此近似分析一致。

### 2.4 配平闭合程度总结

| 条件 | 数值 | 闭合情况 |
|---|---|---|
| 升力平衡所需 α | 1.638° | 与 `aTrim=1.600°` 偏差 −2.3%，近似闭合 |
| 阻力平衡所需 T | 约 6.94 N | `thrTrim=0.5` 提供 8.91 N，偏大 28% |
| 净加速度估算 | 1.97 N / 2.6 kg ≈ 0.76 m/s² | 飞行器将缓慢加速 |

因此，当前配平参数是**近似配平**：`aTrim` 基本满足升力平衡，`thrTrim=0.5` 为“留有推力裕度”的近似油门。精确巡航油门应由所需速度对应的阻力反解，并考虑 SAS 积分项对俯仰/推力的耦合修正。

## 3. C_m0 案例：零迎角俯仰力矩的配平补偿

### 3.1 问题来源

气动模型在零迎角处存在非零俯仰力矩：

```
C_m0 = 0.011
```

其物理来源为机翼弯度/安装角；在零迎角时产生的气动力矩为：

```
M_y,a|_{α=0} = q̄·S_w·c̄·C_m0
             = 352.8 × 0.55 × 0.35 × 0.011
             ≈ 0.747 N·m
```

该力矩方向为正（机头抬头，绕 +y_b）。若不用尾摆角补偿，飞行器将绕 y_b 持续抬头，迎角增大；由于 `C_mα = −0.4 < 0`（静稳定），最终会在某个正迎角处由 `C_m0 + C_mα·α = 0` 平衡，但此时需要推力矢量或其他控制量维持。

### 3.2 用尾摆角平衡 C_m0 的需求

尾电机推力对质心的俯仰力矩（小摆角线性化）为：

```
M_y,t = −b · T_t · sin(δ_t) ≈ −b · T_t · δ_t
```

其中 `b = 0.62 m`，`T_t = 4.455 N`（thrTrim=0.5 时）。令总俯仰力矩为零：

```
−b·T_t·sin(δ_t) + q̄·S_w·c̄·C_m0 = 0
```

解得：

```
sin(δ_t,trim) = (q̄·S_w·c̄·C_m0) / (b·T_t)
              = 0.747 / (0.62 × 4.455)
              = 0.747 / 2.762
              ≈ 0.270
```

```
δ_t,trim ≈ asin(0.270) ≈ 0.274 rad ≈ 15.7°
```

### 3.3 执行机构裕度

摆角限幅：

```
δ_max = P.dMax = 0.436 rad = 25°
```

所需配平摆角占限幅比例：

```
15.7° / 25° ≈ 63 %
```

剩余裕度约 37%。因此，在零迎角情况下，仅靠尾电机摆角即可补偿 `C_m0`，但会占用超过一半的俯仰控制权限。

### 3.4 与 aTrim 的耦合

代码配平迎角 `aTrim = 1.6°` 接近气动自平衡点：

```
α_c = −C_m0 / C_mα = −0.011 / (−0.4) = 0.0275 rad = 1.576°
```

在该迎角下：

```
C_m0 + C_mα·aTrim ≈ 0.011 − 0.4 × 0.027925 ≈ −1.7×10⁻⁴
```

对应的剩余俯仰力矩仅约 `−0.012 N·m`，所需尾摆角约 `0.24°`，几乎为零。这说明 `aTrim` 的选取已经让气动俯仰力矩自平衡，SAS 积分器只需吸收扰动残差。SAS 俯仰积分限幅为 `P.intThMax = 0.5 rad`，远小于所需零迎角补偿量，进一步印证：`aTrim` 是必要设计，不能单纯依赖积分器抵消 `C_m0`。

## 4. 小扰动线性化框架

### 4.1 状态与输入定义

选取机体坐标系小扰动状态（围绕配平直飞状态）：

```
x = [ Δu, Δw, Δq, Δθ, Δv, Δp, Δr, Δφ, Δψ ]^T
```

输入为执行机构修正量：

```
u = [ Δδ_t, Δδ_f, ΔΔω ]^T
```

其中：

- 纵向通道：`Δu, Δw, Δq, Δθ`；输入 `Δδ_t`。
- 横侧向通道：`Δv, Δp, Δr, Δφ, Δψ`；输入 `Δδ_f`（偏航）、`ΔΔω`（滚转）。

### 4.2 纵向模态定性分析

纵向力与力矩在配平点附近线性化：

```
X = X_u Δu + X_w Δw + X_δt Δδ_t
Z = Z_u Δu + Z_w Δw + Z_δt Δδ_t
M = M_u Δu + M_w Δw + M_q Δq + M_δt Δδ_t
```

典型固定翼纵向模态：

| 模态 | 主导状态 | 稳定条件 | 本项目依据 |
|---|---|---|---|
| 短周期 | `Δα`（或 `Δw`）、`Δq` | `M_α < 0`（静稳定）、`M_q < 0`（阻尼） | `C_mα = −0.4 < 0`、`C_mq = −9.0 < 0`，静稳定且阻尼为负 |
| 长周期 / 沉浮 | `Δu`、`|Δθ` | 推力 − 阻力曲线在配平点附近斜率为负 | 当前 `thrTrim=0.5` 推力偏大，长周期表现为缓慢加速而非阻尼；若油门精确匹配阻力，可收敛 |

结论：短周期模态在所列气动导数下是静稳定且有阻尼的；长周期行为受当前近似油门影响，表现为缓慢速度漂移。

### 4.3 横侧向模态定性分析

横侧向力与力矩线性化：

```
Y = Y_v Δv + Y_δf Δδ_f
L = L_v Δv + L_p Δp + L_δω ΔΔω
N = N_v Δv + N_r Δr + N_δf Δδ_f
```

典型固定翼横侧向模态：

| 模态 | 主导状态 | 稳定条件 | 本项目依据 |
|---|---|---|---|
| 荷兰滚 | `Δβ`、`Δr`、`Δφ` 耦合振荡 | `C_nβ > 0`（航向静稳定）、`C_nr < 0`（偏航阻尼） | `C_nβ = 0.12 > 0`、`C_nr = −0.2 < 0`，航向稳定且有阻尼 |
| 滚转收敛 | `Δp` | `C_lp < 0`（滚转阻尼） | `C_lp = −0.5 < 0`，滚转运动快速收敛 |
| 螺旋 | `Δφ` 缓慢收敛/发散 | `C_lβ < 0`（横滚静稳定）抑制发散 | `C_lβ = −0.09 < 0`，有利于螺旋模态稳定 |

滚转控制特殊性：本布局滚转由双发反扭矩差驱动，控制效率 `∂M_x/∂Δω ≈ −2·k_Q·ω₀²` 为负，SAS 符号已按此整定；低油门时反扭矩差基数减小，滚转权限衰减。

### 4.4 适用边界

上述线性化仅在以下条件有效：

- 小迎角、小侧滑角、小摆角；
- 气动模型为线性，未计入失速、压缩性、螺旋桨滑流干扰；
- 推力/阻力平衡近似，长周期分析需配合精确油门；
- 无风、理想执行器（无舵机速率饱和、无安装误差）。

## 5. 数值证据

回归基线定义于 `tests/scenarios.mjs`，采用固定 60 Hz 帧步长驱动 `stepPhysics`，覆盖三条工况：

| 工况 | 描述 | 观察指标 | 结论 |
|---|---|---|---|
| A：配平直飞 10 s | 无额外指令，从 `resetFlightState` 出发 | 速度、迎角、俯仰角 | 速度由 24 m/s 缓慢上升至约 26.2 m/s，姿态角微幅有界 |
| B：俯仰阶跃 8° / 4 s | `δ_t = 8°` 持续 240 帧后撤除 | 俯仰角 θ、角速率 q | 出现俯仰响应峰值，撤除后回到配平附近 |
| C：滚转阶跃 0.25·dwUiMax / 4 s | `Δω = 0.25 × 0.55 = 0.1375` 持续 240 帧后撤除 | 滚转角 φ、滚转速率 p | 滚转角达到峰值后，SAS 保持并回零 |

上述结果属于**演示性数值检查**，仅在所列工况、MODEL-DEFAULT 参数、当前线性气动模型与理想执行器假设下观察到响应有界。不能外推为大机动、失速、悬停或全包线稳定结论。详细验证流程与通过准则见 VER-001。

## 6. 已知限制与未覆盖项

1. **线性气动**：未含大迎角失速、非定常气动力、螺旋桨滑流与机身干扰。
2. **欧拉角奇异**：`eulerFromQuat` 在 `theta = ±90°` 附近出现万向锁；代码未做特殊处理，大俯仰机动可能数值不稳定。
3. **低速 / 悬停**：气动阻尼随动压下降而趋零，姿态稳定完全依赖 SAS；滚转权限受单侧反扭限制，过渡走廊未建模。
4. **理想执行器**：电机为一阶滞后、摆角瞬时到达指令、无舵机带宽与饱和特性。
5. **无风环境**：未引入大气扰动、阵风或风切变。
