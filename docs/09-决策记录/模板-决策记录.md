# 模板 决策记录

## 编号/标题

DEC-xxx-标题

## 状态

提议 / 已接受 / 已废弃 / 已取代

- 「已取代」时需在下方「关联文档」中注明取代者编号。

## 日期

YYYY-MM-DD

## 背景

- 描述决策前的上下文、问题或约束；
- 说明该决策试图解决什么痛点；
- 可引用相关需求、问题记录或前期讨论。

## 决策

- 明确记录所采纳的方案；
- 使用陈述句，避免模糊；
- 若涉及架构、接口或文件位置，给出关键路径。
