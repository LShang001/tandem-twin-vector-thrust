# CTL-002 可控性与失效模式

> 本文从工程实现角度给出「纵列双发矢量推力飞行器」三通道控制效能矩阵、随工况变化的权限包线、主要失效模式及当前处置。源码依据为 `simulations/vector-thrust-lab/src/core/propulsion.mjs` 与 `control.mjs`，理论背景见 [THY-003](../03-理论推导/THY-003-刚体动力学控制分配与模态分析.md)。危害与失效模式的正式注册表由 `docs/registers/危害与失效模式日志.md` 维护（当前尚未创建）。

## 1 控制效能矩阵

### 1.1 小角度线性化

在零摆角附近对 `propulsion.mjs` 六维映射做一阶泰勒展开，得到 3 执行变量 × 3 力矩通道的解析控制效率：

```
M_x ≈ −2·kQ·ω0²·Δω        （滚转，差速主控）
M_y ≈ −b·T_t·δ_t           （俯仰，尾摆角主控）
M_z ≈  a·T_f·δ_f           （偏航，前摆角主控）
```

其中 `ω0 = thr·wMax`，`T_f = T_t = kT·ω0²`（零差速稳态）。写成矩阵形式（忽略耦合小量）：

```
┌ M_x ┐   ┌ 0        0            −2·kQ·ω0² ┐   ┌ δ_t ┐
│ M_y │ = │ 0       −b·kT·ω0²      0        │ · │ δ_f │
└ M_z ┘   └ a·kT·ω0²  0            0        ┘   └ Δω  ┘
```

### 1.2 代入 MODEL-DEFAULT 参数

| 参数 | 值 | 来源 |
|---|---|---|
| kT | 2.2×10⁻⁵ N·s² | MODEL-DEFAULT |
| kQ | 1.6×10⁻⁶ N·m·s² | MODEL-DEFAULT |
| a, b | 0.62 m | MODEL-DEFAULT |
| wMax | 900 rad/s | MODEL-DEFAULT |

在 thr=1（ω0=900 rad/s）时：

```
∂M_y/∂δ_t = −b·kT·ω0² = −0.62 × 2.2×10⁻⁵ × 900² ≈ −11.1 N·m/rad
∂M_z/∂δ_f =  a·kT·ω0² =  0.62 × 2.2×10⁻⁵ × 900² ≈ +11.1 N·m/rad
∂M_x/∂Δω = −2·kQ·ω0² = −2 × 1.6×10⁻⁶ × 900² ≈ −2.59 N·m
```

在 thr=0.5（ω0=450 rad/s）时，上述效率均降至全油门的 1/4。

### 1.3 耦合项讨论

严格映射中的非对角项（交叉耦合）为：

| 耦合项 | 表达式 | 来源 | 量级 |
|---|---|---|---|
| δ_f → M_y | −Q_f·sinδ_f ≈ −kQ·ω_f²·δ_f | 前反扭随前摆角投影到 y 轴 | τ/(T·b) ≈ 10% |
| δ_t → M_z | −Q_t·sinδ_t ≈ −kQ·ω_t²·δ_t | 尾反扭随尾摆角投影到 z 轴 | τ/(T·a) ≈ 10% |
| Δω → M_y, M_z | 二阶小量 | 差速改变推力幅值并耦合到摆角方向 | 可忽略 |

由于 `kQ·ω² ≪ kT·ω²·a`（反扭-推力臂比约 0.1），主对角项占主导，直接映射可行。残余耦合可作为未来前馈补偿项登记。

## 2 可控性包线分析

### 2.1 随油门的权限变化

| 通道 | 全油门效率（thr=1） | 半油门效率（thr=0.5） | 退化规律 |
|---|---|---|---|
| 俯仰 ∂M_y/∂δ_t | −11.1 N·m/rad | −2.77 N·m/rad | ∝ thr² |
| 偏航 ∂M_z/∂δ_f | +11.1 N·m/rad | +2.77 N·m/rad | ∝ thr² |
| 滚转 ∂M_x/∂Δω | −2.59 N·m | −0.65 N·m | ∝ thr² |

俯仰/偏航仍可用是因为摆角力矩 ∝ T ∝ ω0²；滚转退化更严重，因为滚转完全依赖单侧反扭，其力矩仅为单侧 `kQ·ω²` 量级。

### 2.2 低速/悬停场景

本构型**无气动舵面**，俯仰与偏航控制力矩来自推力矢量摆角，与空速无关（只要电机能产生推力）；滚转控制力矩来自差速反扭，亦与空速无关。因此：

- 悬停/低速时俯仰、偏航通道仍有效；
- 滚转通道在低油门时二次衰减，与空速无关；
- 气动阻尼在低速时趋零，姿态稳定更依赖 SAS 角速率反馈。

但需注意：前飞速度增大后，真实螺旋桨推力会随前进比下降（本项目模型未包含此效应，见 [PROP-001](../06-推进与执行机构/PROP-001-推进与摆座模型.md) §6）。

### 2.3 欧拉角 ±90° 奇异

`eulerFromQuat` 采用 `θ = −asin(R13)` 提取俯仰角。当 `|R13| → 1`（俯仰接近 ±90°）时：

- `asin` 自变量饱和到 ±1，姿态提取仍返回有界值；
- 但 `phi` 与 `psi` 出现万向节锁（gimbal lock），`atan2(m23,m33)` 与 `atan2(m12,m11)` 同时退化；
- SAS 比例通道在 90° 附近的控制方向可能发生 180° 跳变，导致失控。

当前仿真未针对大俯仰角做专门保护，仅在 `math.mjs` 中对 `R13` 做 `clamp(...,−1,1)` 防止 NaN。

## 3 失效模式表

| 模式 | 影响 | 可检测性 | 当前处置 | 改进方向 |
|---|---|---|---|---|
| 单前电机失效 | 丧失偏航主控；前反扭消失，尾反扭无对消 → 持续滚转力矩 | 仿真内可直接比较 wf/wt 与指令 | 无自动处置；依赖操作员切断尾电机或改出 | 增加电机转速监控与失效重构逻辑 |
| 单尾电机失效 | 丧失俯仰主控；纵向力矩失衡 | 同上 | 无自动处置 | 同上；考虑推力重新分配或应急气动面 |
| wMax 转速饱和 | 差速大开度时 `ω_f = ω0·√(1+Δω)` 可超过 wMax；当前无钳制 | 当前未检测 | 模型如实计算超 wMax 转速，不报错 | 增加分配后转速饱和保护 + 告警 |
| 积分器极限幅持续饱和 | SAS 积分项顶在 ±intThMax/±intPhiMax，变为恒定偏置 | 可直接读取 intTh/intPhi | 仅依赖限幅，无抗饱和释放 | 增加积分抗饱和（conditional integration 或 back-calculation） |
| 欧拉角 ±90° 奇异 | 万向节锁导致滚转/偏航通道方向耦合 | 检测 `|theta|` 接近 π/2 | 无保护 | 大姿态角切换四元数直接控制律 |
| 传感器失效 | 仿真内状态直接可用，无传感器模型 | 不适用 | 不适用 | 增加传感器模型与状态估计器（如卡尔曼滤波） |

上述失效模式的正式登记、风险评级与处置跟踪应在 `docs/registers/危害与失效模式日志.md` 中维护。当前该注册表尚未创建，建议在阶段 9「既有文档勘误」中补齐。

## 6 小结

- 三通道主控效率对角占优，直接映射控制架构在概念级仿真中可行；
- 滚转通道效率最低且随油门平方衰减，是整机可控性瓶颈；
- 单电机失效、转速饱和、积分饱和、大俯仰奇异为当前主要失效/退化模式；
- 所有参数状态均为 MODEL-DEFAULT，可控性包线分析为解析估算，需经台架/飞行验证更新。
