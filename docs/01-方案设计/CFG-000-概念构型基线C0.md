# CFG-000 概念构型基线 C0

> 本篇给出「纵列双发矢量推力飞行器」C0 概念构型的整机定义：布局、质量特性、推进方式、控制机理、气动参考量与已知边界。参数全部来自 `models/aircraft-model.json`，当前来源标记为 `MODEL-DEFAULT`。

---

## 1 构型概述与布局

本概念机采用**前后纵列双电机 + 正交单轴矢量摆座 + 反向旋转螺旋桨**构型。机体系约定见 [MOD-002](../04-数学建模/MOD-002-坐标系与符号约定.md)：x 轴指向前，y 轴指向右，z 轴指向下方（NED 体系），原点在质心。

```mermaid
flowchart LR
    accTitle 纵列双发布局与控制轴示意图
    accDescr 展示前电机位于机头 +a 处、绕 z_b 摆动主控偏航；尾电机位于机尾 -b 处、绕 y_b 摆动主控俯仰；滚转由双发差速反扭差实现。

    subgraph 侧视图 x-z 平面
        FM[前电机<br/>拉力式 CW<br/>绕 z_b 摆动]
        CG((质心))
        RM[尾电机<br/>推进式 CCW<br/>绕 y_b 摆动]
        FM -- "a = 0.62 m" --> CG
        CG -- "b = 0.62 m" --> RM
    end
```

关键几何事实：

- 前电机力臂 `a = 0.62 m`（质心到前电机中心），尾电机力臂 `b = 0.62 m`；
- 零摆角时两电机推力线均通过质心，静态推力不产生俯仰/偏航力矩；
- 渲染几何中电机位置约为 `±1.78 m`，**仅用于可视化**，不进入动力学计算（参见 [MOD-001](../04-数学建模/MOD-001-六自由度仿真模型规范.md) 与 [MOD-002](../04-数学建模/MOD-002-坐标系与符号约定.md)）。

## 3 推进构型

### 3.1 前后电机角色

| 电机 | 安装位置 | 旋转方向 | 摆轴 | 主控通道 | 备注 |
|------|----------|----------|------|----------|------|
| 前电机 | 机头 +a | 正转 CW | 绕 z_b 单轴 | 偏航 | 拉力式 |
| 尾电机 | 机尾 -b | 反转 CCW | 绕 y_b 单轴 | 俯仰 | 推进式 |

### 3.2 旋向选择理由

前电机 CW、尾电机 CCW，使零差速等转速时两机反扭矩大小相等、方向相反，机体净滚转力矩近似为零。该设计把滚转通道从俯仰/偏航中**解耦**出来，滚转仅由双发差速产生的反扭差驱动，与常规共轴直升机扭矩差原理一致。

### 3.3 摆轴选择逻辑

- **前电机绕 z_b 摆动**：推力在机体 x-y 平面内偏转，产生侧向力 Fy 与偏航力矩 Mz = a·Tf·sinδf，与常规方向舵效应同向；
- **尾电机绕 y_b 摆动**：推力在机体 x-z 平面内偏转，产生垂向力 Fz 与俯仰力矩 My = -b·Tt·sinδt；
- 两摆轴正交，使俯仰与偏航主控力矩分别由独立执行机构产生，通道间仅存在二阶耦合项（见 §4）。

## 4 控制机理

### 4.1 推力与反扭矩模型

```
Tf = kT · ωf²          Tt = kT · ωt²
Qf = kQ · ωf² + Jp · ω̇f      Qt = kQ · ωt² + Jp · ω̇t
```

差速开方分配（保持 ωf² + ωt² ≈ 2·ω0²，总推力在一阶近似下不变）：

```
ω0 = thr · wMax
ωf = ω0 · sqrt(max(0, 1 + Δω))
ωt = ω0 · sqrt(max(0, 1 - Δω))
```

### 4.2 六维力 / 力矩映射（机体系）

```
Fx = Tf·cosδf + Tt·cosδt
Fy = Tf·sinδf
Fz = -Tt·sinδt

Mx = -Qf·cosδf + Qt·cosδt
My = -b·Tt·sinδt - Qf·sinδf
Mz =  a·Tf·sinδf - Qt·sinδt
```

### 4.3 三通道映射表

| 通道 | 主控输入 | 主控力矩项 | 控制效率符号 | 零位特性 |
|------|----------|------------|--------------|----------|
| 俯仰 | 尾摆角 δt | My ≈ -b·Tt·sinδt | ∂My/∂δt < 0 | δt=0 时 My=0 |
| 偏航 | 前摆角 δf | Mz ≈ a·Tf·sinδf | ∂Mz/∂δf > 0 | δf=0 时 Mz=0 |
| 滚转 | 差速 Δω | Mx ≈ Qt - Qf（小摆角） | ∂Mx/∂Δω < 0 | Δω=0 时 Mx≈0 |

残余耦合：

- 偏航摆动 δf 会通过 `-Qf·sinδf` 向俯仰通道注入扰动；
- 俯仰摆动 δt 会通过 `-Qt·sinδt` 向偏航通道注入扰动；
- 上述耦合在小摆角下为二阶小量，可在后续控制分配中以前馈方式补偿（参见 [CTL-002](../05-控制与分配/CTL-002-可控性与失效模式.md)）。

## 5 气动布局

本阶段采用**线性小迎角气动导数模型**，气动力 / 力矩由风轴系转换至机体系。详细公式见 [MOD-001](../04-数学建模/MOD-001-六自由度仿真模型规范.md)。

```
V   = max(sqrt(u² + v² + w²), vMin)
α   = atan2(w, u)
β   = asin(v / V)
q̄   = 0.5 · rho · V²
CL  = CLa · α
L   = q̄ · Sw · CL
D   = q̄ · Sw · (CD0 + CDk · CL²)
Y   = q̄ · Sw · CYb · β
```

气动矩：

```
pH = p·bspan / (2V) ,  qH = q·cbar / (2V) ,  rH = r·bspan / (2V)

Mx,a = q̄·Sw·bspan·(Clb·β + Clp·pH)
My,a = q̄·Sw·cbar·(Cm0 + Cma·α + Cmq·qH)
Mz,a = q̄·Sw·bspan·(Cnb·β + Cnr·rH)
```

当前气动模型仅覆盖小迎角、无侧滑或小刚体角速率范围，未包含失速、非定常效应、螺旋桨滑流及风扰。

## 6 已知限制与边界

| 类别 | 限制说明 | 影响 |
|------|----------|------|
| 执行机构 | 理想执行机构：摆角瞬时到位、无速率限制、无舵机动态 | 高频姿态指令下的相位滞后未建模 |
| 转速保护 | 差速大开度时单机目标转速可超过 wMax，当前无饱和保护 | 大 Δω 下推进模型可能超出物理边界 |
| 滚转效能 | 低油门时总推力小，反扭差幅值低，滚转控制能力退化 | 低速 / 低油门机动受限 |
| 渲染几何 | 三维模型电机位置约 ±1.78 m，与动力学力臂 a=b=0.62 m 不一致 | 仅视觉效果，不影响力/力矩计算 |
| 大气模型 | 无风、无湍流、无阵风扰动；密度恒定 rho=1.225 kg/m³ | 仅适用于平静大气概念研究 |
| 气动模型 | 线性小迎角导数；无失速、滑流、非定常气动 | 大迎角 / 大侧滑角结果不可信 |
| 硬件范围 | 仅软件仿真，无传感器、飞控、结构或台架 | 结果不构成飞行性能承诺 |
| 姿态积分 | 显式欧拉 + 四元数右乘，子步上限 4 ms | 仅满足概念演示需求，非高保真积分器 |

## 7 基线状态声明

- **基线代号**：C0（概念构型基线）。
- **参数状态**：全部参数当前为 `MODEL-DEFAULT`，未经台架标定或飞行辨识，置信度标记为 `low`。
- **模型成熟度**：概念级 / 演示级，用于控制律结构研究、通道机理演示与教学，不用于飞行性能预测。
- **变更控制**：任何涉及质量、惯量、力臂、气动导数、推进参数或摆轴定义的变更，须通过 [GOV-002](../00-项目治理/GOV-002-信息与配置管理.md) 配置管理流程，更新 `models/aircraft-model.json`，并重新运行 `tools/sync-params.py` 与回归测试。
