# CONOPS-001 运行概念

> 本篇说明「纵列双发矢量推力飞行器」Web 六自由度仿真平台 `simulations/vector-thrust-lab` 的运行概念（ConOps）：用户是谁、如何操作、典型场景、运行环境及扩展边界。技术细节见 [CFG-000](CFG-000-概念构型基线C0.md) 与 [MOD-001](../04-数学建模/MOD-001-六自由度仿真模型规范.md)。

---

## 1 平台定位

本仿真平台是项目的**当前唯一实现载体**，定位为：

- **概念验证工具**：在 C0 构型下快速验证三通道控制分配、SAS 增稳与气动阻尼的定性行为；
- **控制律研究平台**：为后续控制架构（[CTL-001](../05-控制与分配/CTL-001-控制架构与SAS.md) / [CTL-002](../05-控制与分配/CTL-002-可控性与失效模式.md)）提供可复现的数值试验环境；
- **教学演示工具**：通过交互式三维可视化展示纵列双发、正交单轴摆座、差速反扭滚转等非常规布局的机理。

平台当前为**纯软件仿真**，不包含任何飞行硬件、传感器模型或真实嵌入式控制代码。

## 2 用户角色

| 角色 | 关注点 | 典型操作 |
|------|--------|----------|
| 研究者 | 控制律结构、通道耦合、配平与稳定性 | 修改 `models/aircraft-model.json`，运行回归测试，对比不同参数下的响应 |
| 学习者 | 矢量推力布局机理、六自由度刚体动力学 | 使用 UI 滑块操纵飞行器，观察 HUD 与六维力矩方程面板 |

两个角色共享同一入口页面，区别仅在于是否深入代码与参数源。

## 3 运行模式

仿真页面提供以下运行模式与开关，全部通过浏览器 DOM 控件操作。

### 3.1 四通道手动操纵

| 输入 | 控件 ID | 范围 | 物理含义 |
|------|---------|------|----------|
| 总油门 thr | `s-thr` / `v-thr` | 0% ~ 100% | 指令总转速基准 ω0 = thr · wMax |
| 俯仰摆角 δt | `s-dt` / `v-dt` | -25° ~ +25° | 尾电机绕 y_b 摆动 |
| 偏航摆角 δf | `s-df` / `v-df` | -25° ~ +25° | 前电机绕 z_b 摆动 |
| 差速 Δω | `s-dw` / `v-dw` | -30% ~ +30%（UI） | 映射为 ±dwUiMax（±0.55）的差速指令 |

### 3.2 功能开关

| 开关 | 控件 ID | 默认状态 | 作用 |
|------|---------|----------|------|
| 增稳 SAS | `b-sas` | 开 | 启用角速率 / 姿态角 / 积分反馈（见 [CTL-001](../05-控制与分配/CTL-001-控制架构与SAS.md)） |
| 气动力 | `b-aero` | 开 | 启用线性气动模型；关闭后仅保留推力与重力 |
| 复位 | `b-reset` | - | 状态回到配平初始条件（vTrim / aTrim / thrTrim） |

### 3.3 演示序列

点击演示按钮后，仿真自动注入周期指令；再次点击同一按钮停止。

| 序列 | 控件 ID | 指令形式 | 周期 |
|------|---------|----------|------|
| 俯仰演示 | `b-pitch` | δt = 18° · sin(2πt / 3.2 s) | 3.2 s |
| 偏航演示 | `b-yaw` | δf = 18° · sin(2πt / 3.2 s) | 3.2 s |
| 滚转演示 | `b-roll` | Δω = 0.28 · sin(2πt / 3.2 s) | 3.2 s |
| 综合演示 | `b-cine` | 12 s 分段：油门爬升 → 俯仰 → 偏航 → 滚转 | 12 s |

页面加载 900 ms 后自动启动综合演示（`cine`），用于首次访问时展示整机运动。

### 3.4 辅助功能

| 功能 | 控件 ID | 说明 |
|------|---------|------|
| 文档中心 | `b-info` / `modal` | 弹出模型方程与三通道机理说明；链接 `./docs/index.html` 打开理论文档 |
| 主题切换 | `b-theme` | 浅色 / 深色主题切换 |

## 4 典型使用场景

### 4.1 配平直飞观察

用户启动页面后保持默认油门 44.135%，不操作滑块，观察飞行器在 `aTrim=1.627°`、`dtTrim=-0.650°`、`vTrim=24 m/s` 的纵向配平参考点附近飞行。尾摆滑块零位表示相对配平增量为零。用途：验证纵向三方程残差闭合、横航向残差量级以及 SAS 对小姿态扰动的衰减；该工况不是六自由度全配平。

### 4.2 姿态阶跃响应

用户手动给出单一摆角阶跃（如 δt = 10°），随后保持不动，观察俯仰角 θ 与角速率 q 的瞬态。用途：在「气动力开 / 关」两种状态下对比气动阻尼对响应有界性的贡献。

### 4.3 通道解耦演示

依次启动俯仰、偏航、滚转三个单通道演示，配合方程面板观察 Mx / My / Mz 的主导项。用途：直观理解正交摆轴如何把俯仰、偏航、滚转分配到不同执行机构。

### 4.4 失效与边界探索

用户关闭 SAS 或关闭气动力、降低油门后施加滚转差速，观察低油门下滚转效能退化及开环不稳定趋势。用途：识别 C0 构型的控制边界，为后续失效模式分析（[CTL-002](../05-控制与分配/CTL-002-可控性与失效模式.md)）提供感性认识。

## 5 运行环境

### 5.1 启动方式

方式一（推荐）：

```bat
simulations\vector-thrust-lab\启动.bat
```

方式二（手动）：

```bash
cd simulations/vector-thrust-lab
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

必须通过 HTTP 访问；直接双击 `index.html` 会因 ESM / import map 安全策略而失败。

### 5.2 依赖

| 依赖 | 版本 / 说明 |
|------|-------------|
| Python 3 | 仅用于静态文件服务 |
| 现代浏览器 | 支持 ES Module、import map、WebGL |
| Three.js | r170，位于 `vendor/three-r170/`（MIT） |
| Node.js | 用于运行 `node --test tests/` 单元与回归测试 |

### 5.3 自动化测试

```bash
cd simulations/vector-thrust-lab
node --test tests/
```

测试覆盖核心模块：数学工具、状态初始化、SAS 增稳、推进系统、气动力、六自由度积分及三条回归基线（配平直飞 / 俯仰阶跃 / 滚转阶跃）。

## 6 约束与未来扩展

### 6.1 当前约束

- 仅软件仿真，无飞行硬件；
- 参数全部为 `MODEL-DEFAULT`，未经验证；
- 气动为线性小迎角模型，不覆盖失速与大机动；
- 执行机构为理想模型，无舵机速率 / 带宽限制；
- 无传感器噪声、无通信延迟、无嵌入式软件。

### 6.2 未来扩展方向（仅登记，不展开）

未来若从仿真向实物迭代，潜在路径包括：

1. **台架试验**：电机 + 电调 + 六分量天平，标定 kT、kQ、Jp、tauM；
2. **系留飞行**：验证三通道响应与 SAS 参数整定；
3. **自由飞原型**：引入 IMU、GPS、飞控计算机与机载控制律。

上述任一路径涉及范围、参数来源或安全承诺的变更时，须按 [GOV-002](../00-项目治理/GOV-002-信息与配置管理.md) 更新项目范围与基线。
