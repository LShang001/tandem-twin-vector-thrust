矢量推力飞行动力学实验室 — 理论分析与建模文档

对象布局：纵列双发矢量推力飞行器 —— 机头前电机（正转，绕 z_b 轴单轴摆动，偏航通道）、机尾电机（反转，绕 y_b 轴单轴摆动，俯仰通道）、差速控制滚转。本文档给出从单电机矢量座到整机六自由度飞行动力学的完整推导、气动与配平分析、增稳控制律设计及数值验证结果。

1. 坐标系与符号约定

坐标系 定义
惯性系 {I} 地面参考系，z 向下（NED 约定）
机体系 {B} x_b 指机头，y_b 指右翼，z_b 指向下，原点在质心
摆座/电机系 {M} 随电机摆动，x_m 沿电机轴（推力方向）
符号 含义 符号 含义
δ_f 前电机摆角（绕 z_b，偏航） δ_t 尾电机摆角（绕 y_b，俯仰）
ω_f, ω_t 前/尾电机转速 Δω 差速指令
T = k_T·ω² 螺旋桨推力 τ = k_Q·ω² + J_p·ω̇ 传给机体的反扭矩
a, b 前/尾电机到质心的距离 p, q, r 机体系角速度分量
q̄ = ½ρV² 动压 α, β 迎角 / 侧滑角

约定：两电机轴线在零摆角时均过质心（推力线通过质心，无静态力矩）；前电机为拉力式，尾电机为推进式，旋向相反。

2. 推力矢量映射

单轴摆动即绕固定轴的旋转。尾电机绕 y_b 摆 δ_t、前电机绕 z_b 摆 δ_f，旋转矩阵：

        ⎡ cosδ  0  sinδ ⎤              ⎡ cosδ  -sinδ  0 ⎤
R_y(δ) = ⎢  0    1   0   ⎥      R_z(δ) = ⎢ sinδ   cosδ  0 ⎥
        ⎣ -sinδ 0  cosδ ⎦              ⎣  0      0    1 ⎦

推力在电机系中沿 x_m，T = [T, 0, 0]ᵀ。投影到机体系：

尾电机:  F_t = R_y(δ_t)·[T_t,0,0]ᵀ = [T_t·cosδ_t,  0,  -T_t·sinδ_t]ᵀ
前电机:  F_f = R_z(δ_f)·[T_f,0,0]ᵀ = [T_f·cosδ_f,  T_f·sinδ_f,  0]ᵀ

对质心的力矩为安装位置矢量与推力的叉乘 M = r × F。取 r_f = [+a, 0, 0]、r_t = [−b, 0, 0]：

尾电机俯仰力矩:  M_y = -b·T_t·sinδ_t ≈ -b·T_t·δ_t   （小摆角线性化）
前电机偏航力矩:  M_z = +a·T_f·sinδ_f ≈ +a·T_f·δ_f

物理图像：零摆角时推力线过质心、力矩为零；摆动后推力线偏转，等效力臂为 a·sinδ 或 b·sinδ，控制力矩在小摆角范围内与 δ 近似线性——对控制分配非常友好。

3. 反扭力矩、差速滚转与陀螺效应

3.1 反扭力矩的完整表达

螺旋桨旋转时，真正经电机定子传给机体的是电磁扭矩，除气动阻力矩外还包含转子加速惯性项：

τ = k_Q·ω² + J_p·ω̇

机动加减速时第二项不可忽略。反扭矩沿各自电机轴，随机座一起摆动：

前电机（正转）:  M_Qf = R_z(δ_f)·[-τ_f,0,0]ᵀ = [-τ_f·cosδ_f, -τ_f·sinδ_f, 0]ᵀ
尾电机（反转）:  M_Qt = R_y(δ_t)·[+τ_t,0,0]ᵀ = [+τ_t·cosδ_t,  0, -τ_t·sinδ_t]ᵀ

3.2 滚转控制机理（关键）

零摆角时前后反扭矩互消（M_x = τ_t − τ_f = 0）。差速打破对消产生滚转力矩——注意机理：前后电机都在纵轴上、推力均过质心，推力差对质心不产生力矩，滚转完全由反扭矩差驱动，与共轴直升机扭矩差控制偏航同一原理：

ΔM_x = k_Q·(ω_t² - ω_f²) + J_p·(ω̇_t - ω̇_f)

分配律取 ω_f = ω₀√(1+Δω)、ω_t = ω₀√(1−Δω)，保持 ω_f²+ω_t² 为常数，滚转机动一阶近似下不扰动总推力。

3.3 摆动引入的交叉耦合

  • 偏航摆动 δ_f → 俯仰扰动 −τ_f·sinδ_f
  • 俯仰摆动 δ_t → 偏航扰动 −τ_t·sinδ_t

因 τ ≪ T·力臂（通常差一个数量级），耦合为二阶小量，且结构已知，可在混控中前馈补偿。

3.4 陀螺力矩

双转子角动量（前正后反，沿各自摆座轴）：

h = J_p·ω_f·[cosδ_f, sinδ_f, 0]ᵀ − J_p·ω_t·[cosδ_t, 0, −sinδ_t]ᵀ

其对机体的作用 −ω×h 进入转动方程。双桨反转使零摆角时姿态速率引起的陀螺耦合天然互消，这是该布局的附带红利。

4. 推力通道六维方程汇总

F_x = T_f·cosδ_f + T_t·cosδ_t        M_x = -τ_f·cosδ_f + τ_t·cosδ_t
F_y = T_f·sinδ_f                      M_y = -b·T_t·sinδ_t - τ_f·sinδ_f
F_z = -T_t·sinδ_t                     M_z =  a·T_f·sinδ_f - τ_t·sinδ_t

控制效率矩阵（小摆角，用于混控/分配）：

∂M_y/∂δ_t = -b·T_t·cosδ_t      ∂M_z/∂δ_f = a·T_f·cosδ_f      ∂M_x/∂Δω ≈ -2·k_Q·ω₀²
通道 操纵量 主控项 带宽限制 备注
俯仰 尾摆角 δ_t −b·T_t·sinδ_t 舵机 权限随油门增长
偏航 前摆角 δ_f +a·T_f·sinδ_f 舵机 权限随油门增长
滚转 差速 Δω k_Q·(ω_t²−ω_f²) 电机转速动态 权限仅为单侧反扭量级，低油门显著衰减

5. 刚体六自由度动力学

5.1 平动方程（机体系）

m·v̇ = F_推力 + F_气动 + m·g_b − m·(ω × v)

g_b 为重力矢量在机体系中的分量（由姿态四元数变换），ω×v 为体轴旋转引起的科氏项。

5.2 转动方程（含旋转部件）

I·ω̇ = M_推力 + M_气动 − ω×(I·ω) − ω×h_转子

ω×(I·ω) 是刚体欧拉方程的陀螺耦合项，ω×h_转子 是双转子角动量的进动项——两者在大角速度机动时不可忽略。

5.3 姿态运动学：四元数右乘形式

机体系角速度对应的姿态微分方程为 q̇ = ½·q ⊗ ω̃（ω̃ 为纯四元数，右乘）。若误用世界系形式 q̇ = ½·ω̃⊗q 而代入机体系角速度，小姿态角下近似成立，大姿态角下响应方向错乱——这是数值实现中的常见陷阱。

6. 气动模型（风轴系）

L = q̄·S·C_Lα·α          D = q̄·S·(C_D0 + k·C_L²)          Y = q̄·S·C_Yβ·β
M_x,a = q̄·S·b̄·(C_lβ·β + C_lp·p̂)      p̂ = p·b̄/(2V)
M_y,a = q̄·S·c̄·(C_m0 + C_mα·α + C_mq·q̂)   q̂ = q·c̄/(2V)
M_z,a = q̄·S·b̄·(C_nβ·β + C_nr·r̂)      r̂ = r·b̄/(2V)

阻尼导数的物理意义（本项目核心问题）：摆角产生角加速度，角速度若无阻尼确实会无限积分。真实物理中的有界化机制有二：

  1. 气动阻尼：C_mq / C_lp / C_nr 提供与角速率成正比的阻尼力矩，α/β 建立后 C_mα 等静稳定力矩进一步提供恢复——阶跃响应收敛到新的平衡点；
  2. 闭环增稳：气动阻尼 ∝ q̄，低速/悬停时趋零，此时姿态稳定完全由 SAS 承担。本仿真提供”气动力忽略”开关，可复现”角速度自由积分”的裸物理。

7. 配平与稳定性分析

7.1 配平条件

巡航配平需同时满足：T = D、L ≈ m·g、合力矩为零。纵向力矩平衡：

q̄·S·c̄·(C_m0 + C_mα·α_trim) = 0   →   α_trim = −C_m0/C_mα

7.2 C_m0 的必要性：一次真实的发散案例

建模过程中实测发现：若 C_m0 = 0（无配平弯度），飞机从近似配平点出发后持续俯冲发散。机理链条：

  • 配平迎角需由推力矢量持续顶住 C_mα·α 的静稳定力矩；
  • 所需控制矩 ∝ q̄ 增长，摆座权限固定 → 可保持的 α ∝ 1/q̄ 衰减；
  • 于是升力 L = q̄·S·C_Lα·α 与速度解耦、恒小于重力；
  • 短周期稳定性使机头跟随航迹下压，迎角无法重建 → 加速俯冲。

这正是控制权限 vs 静稳定度的配平问题，与真实飞机一致。解法亦与真实飞机相同：通过机翼弯度/安装角引入 C_m0，使巡航点气动矩自平衡，增稳系统只承担扰动抑制。取 C_m0 = 0.011（α_trim ≈ 1.6° @ V ≈ 26 m/s）后全包线稳定。

8. SAS 增稳控制律

结构：角速率阻尼 + 姿态角比例/积分反馈（积分消除常值配平误差，带抗饱和限幅）：

δ_t ← δ_t + k_q·q + k_θ·θ + k_i·∫θ        （俯仰，∂M_y/∂δ_t < 0）
δ_f ← δ_f − k_r·r                          （偏航，∂M_z/∂δ_f > 0）
Δω  ← Δω  + k_p·p + k_φ·φ + k_iφ·∫φ       （滚转，∂M_x/∂Δω < 0）

反馈极性必须按各通道控制效率符号整定：效率为负的通道（俯仰、滚转）取正号，效率为正的通道（偏航）取负号，否则构成正反馈——调试中曾因此出现姿态发散，符号校验后收敛。执行量经 ±25° 摆角饱和与差速限幅。

11. 局限与后续方向

  • 气动模型为线性小迎角模型，未含大迎角失速、非定常效应与螺旋桨滑流干扰；
  • 滚转权限受单侧反扭量级限制，低油门时滚转能力弱（物理固有，建议大机动工况配合气动舵面）；
  • 单电机/单舵面失效无余度；
  • 后续可扩展：控制分配在线优化、LQR/模型预测姿态控制、过渡走廊（悬停—巡航）包线仿真。
